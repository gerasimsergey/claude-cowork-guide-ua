# Шпаргалка Claude Cowork (Cheatsheet)

🌐 **Мови**: [English](cheatsheet.md) | [Français](cheatsheet.fr.md) | [Українська 🇺🇦](cheatsheet.uk.md)

Стислий довідник з найважливіших команд та принципів роботи для швидкого старту.

---

## 🚀 Формула ідеального промпту (CTOC)

| Блок | Що це означає | Приклад |
| :--- | :--- | :--- |
| **Context** | Світ навколо завдання | "Я готую звіт для MacPlus Software..." |
| **Task** | Що саме зробити | "...витягни суми ПДВ з усіх чеків..." |
| **Output** | Бажаний формат | "...збережи як таблицю CSV." |
| **Constraints** | Правила та обмеження | "Не змінюй дати. Використовуй валюту UAH." |

---

## 📁 Робота з файлами

- **Читання**: "Прочитай файл [ім'я_файлу]"
- **Аналіз папки**: "Проскануй папку [шлях] та склади список файлів"
- **Організація**: "Перемісти всі PDF у папку /Documents"
- **Створення**: "Створи новий файл [назва.md] з таким вмістом..."

---

## 💡 Поради для ефективності

1. **"Спочатку план"**: Завжди просіть Claude: *"Спочатку напиши план дій, а я його затверджу"*.
2. **Пакетна обробка**: Замість обробки файлів по одному, кажіть: *"Опрацюй всі 20 файлів у цій папці одночасно"*.
3. **Менше слів**: Claude Cowork краще розуміє короткі та чіткі інструкції.
4. **Контекстне вікно**: Якщо сесія стала занадто довгою (Claude почав "забувати"), почніть нову сесію, надавши їй підсумок попередньої.

## 🛠️ Корисні інструменти (MCP)

- `brave_search` — для пошуку свіжої інформації в інтернеті.
- `google_drive` — для доступу до хмарних документів.
- `slack` — для надсилання звітів команді.
