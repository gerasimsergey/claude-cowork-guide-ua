# Глосарій Claude Cowork

🌐 **Мови**: [English](glossary.md) | [Français](glossary.fr.md) | [Українська 🇺🇦](glossary.uk.md)

Цей глосарій містить ключові терміни, що використовуються в посібнику та під час роботи з Claude Cowork.

---

### А
- **Агент (Agent)** — ШІ-модель (як Claude), яка здатна виконувати завдання автономно, використовуючи доступні інструменти (читання файлів, запуск команд).
- **Артефакт (Artifact)** — Візуальний результат роботи Claude (текст, код, діаграма), який відображається в окремому вікні для зручного перегляду та редагування.

### В
- **Воркфлоу (Workflow)** — Покрокова інструкція або набір дій для виконання конкретного бізнес-завдання за допомогою ШІ.

### Г
- **Голос бренду (Tone of Voice)** — Стиль спілкування Claude, адаптований під вашу компанію (наприклад: професійний, дружній, лаконічний).

### Д
- **Декомпозиція (Decomposition)** — Процес розбиття складного завдання на менші, простіші етапи.

### І
- **Інвойс (Invoice)** — Рахунок-фактура на оплату товарів або послуг.
- **Ітерація (Iteration)** — Один цикл процесу уточнення або покращення результату.

### К
- **Контекст (Context)** — Вся інформація, яку Claude знає про ваше завдання в межах однієї сесії (файли, минулі повідомлення).
- **Кошторис (Quote / Estimate)** — Попередня оцінка вартості та обсягу робіт для клієнта.

### М
- **МСР (Model Context Protocol)** — Протокол, який дозволяє Claude підключатися до зовнішніх інструментів та даних (Google Search, Slack, бази даних).

### П
- **Промпт (Prompt)** — Текстовий запит, який ви надсилаєте Claude для виконання завдання.
- **План виконання (Action Plan)** — Перелік кроків, які Claude пропонує зробити перед початком роботи.

### С
- **Системний промпт (System Prompt)** — Набір постійних інструкцій, які визначають поведінку Claude (його "пам'ять" та правила).
