## Діагностичне дерево рішень

Використовуйте цю схему для ідентифікації проблеми:

```
Що відбувається?
│
├─ Cowork не запускається або не відображається
│   └─ Перейдіть до: § ПРОБЛЕМИ НАЛАШТУВАННЯ
│
├─ "VM connection timeout" або "workspace failed"
│   └─ Перейдіть до: § ПРОБЛЕМИ З VM ТА ПІДКЛЮЧЕННЯМ (ймовірно VPN)
│
├─ Cowork не має доступу до моїх файлів
│   └─ Перейдіть до: § ПРОБЛЕМИ З ПРАВАМИ ДОСТУПУ
│
├─ Завдання зупиняється посеред виконання або "context limit"
│   └─ Перейдіть до: § ПРОБЛЕМИ ВИКОНАННЯ
│
├─ Результат (вивід) неправильний або неочікуваний
│   └─ Перейдіть до: § ПРОБЛЕМИ З РЕЗУЛЬТАТАМИ
│
├─ Проблеми з інтеграцією браузера/Chrome
│   └─ Перейдіть до: § ПРОБЛЕМИ З БРАУЗЕРОМ
│
└─ Щось інше
    └─ Перейдіть до: § ЗАГАЛЬНІ ПОРАДИ
```

## Короткий довідник помилок

| Повідомлення | Ймовірна причина | Швидке рішення |
|--------------|--------------|-----------|
| `Failed to start Claude's workspace — VM connection timeout after 60 seconds` | Активний VPN | Відключіть VPN → повторіть |
| `Chrome native messaging host not found` | Помилка розширення | Ручне встановлення хоста (див. нижче) |
| `Context limit reached` (біля ~165K) | Системні витрати | Розбийте завдання на менші порції |
| `Access denied — path outside allowed directories` | Немає доступу до папки | Надайте доступ до папки повторно |
| `Session terminated unexpectedly` | Сплячий режим/фон | Тримайте додаток активним, вимкніть сон |
| `Cannot connect to Chrome` | Відсутні дозволи | Надайте дозвіл Accessibility (Доступність) |

## Проблеми налаштування

### Опція Cowork не відображається

**Симптоми**:
- У селекторі режимів немає Cowork.
- Перемикач функцій відсутній у налаштуваннях.

**Рішення**:
1. **Перевірте підписку**: Має бути тариф Pro або Max.
2. **Оновіть додаток**: Claude Desktop → Check for Updates.
3. **Перезапустіть додаток**: Повністю закрийте (Cmd+Q на Mac, Alt+F4 на Windows) та запустіть знову.
4. **Очистіть кеш**: 
   - macOS: Видаліть `~/Library/Application Support/Claude/` та перезапустіть.
   - Windows: Видаліть `%APPDATA%\Claude\` та перезапустіть.

## Проблеми з VM та підключенням

### "VM Connection Timeout" (Проблема з VPN)

**Текст помилки**:
```
Failed to start Claude's workspace — VM connection timeout after 60 seconds
```

**Причина**: VPN-програми створюють конфлікти маршрутизації з внутрішньою мережею віртуальної машини Cowork. Це **проблема №1**, про яку повідомляють користувачі.

**Рішення**:
1. **Повністю відключіть VPN** перед запуском Cowork.
2. Якщо VPN необхідний: Використовуйте роздільне тунелювання (split tunneling), щоб виключити Claude.
3. Спробуйте мобільний хотспот як тимчасове рішення.

## Проблеми з правами доступу

### "Cannot access folder" (Немає доступу до папки)

**Рішення для macOS**:
1. Відкрийте **System Settings** → **Privacy & Security**.
2. Перейдіть до **Files and Folders**.
3. Знайдіть **Claude** або **Claude Desktop**.
4. Переконайтеся, що доступ до вашої робочої папки дозволено.

**Рішення для Windows**:
1. Правою кнопкою на папку робочого простору → **Properties** → вкладка **Security**.
2. Переконайтеся, що ваш обліковий запис має "Full Control" (Повний доступ).
3. Перевірте, чи не блокує додаток антивірус або Брандмауер Windows.

## Проблеми виконання

### Завдання зупиняється посеред роботи

| Причина | Рішення |
|-------|----------|
| **Ліміт контексту** | Розбийте завдання на менші частини |
| **Таймаут** | Тримайте додаток активним, зменште обсяг завдання |
| **Додаток у фоні** | Тримайте вікно Claude Desktop відкритим і активним |
| **Файли заблоковані** | Закрийте інші програми, що використовують ці файли |

### Ліміт контексту вичерпано завчасно

**Чому це стається**: Системні накладні витрати (визначення інструментів, правила безпеки) споживають ~25-35К токенів ще до початку вашого завдання. Ефективний ліміт — близько 165К токенів, а не 200К.

**Стратегії вирішення**:
- Обробляйте по 10-20 файлів за сесію, а не по 50+.
- Починайте нову розмову, щоб очистити контекст.
- Зберігайте проміжні результати в текстові файли.

## Проблеми з результатами

### Формули Excel не працюють

**Симптоми**: Формули відображаються як текст або видають помилку `#NAME?`.

**Рішення**: 
Вкажіть у промпті: "Використовуй крапку з комою як розділювач у формулах (для європейського регіону)" або "Використовуй кому (для США)". Клітинка також може бути відформатована як текст; змініть формат на "Числовий".

## Проблеми з браузером

### Інтеграція з Chrome не працює

1. Переконайтеся, що встановлено оригінальний Chrome (не Chromium).
2. Надайте системні дозволи: **System Settings → Privacy & Security → Accessibility (Доступність)** → Додайте Claude Desktop.
3. Тестуйте: Попросіть Cowork "Open Chrome and search for 'test'".

---

## Специфічні проблеми Windows

1. **Запуск від імені адміністратора**: Спробуйте запустити Claude Desktop з правами адміністратора для первинного налаштування.
2. **Виключення антивірусу**: Додайте Claude Desktop до списку виключень (Exclusions) у Windows Defender або вашому антивірусі.
3. **Шляхи до папок**: Уникайте розміщення робочого простору в системних папках (наприклад, `C:\Windows\`). Краще створіть папку безпосередньо в корені диска або в профілі користувача.
