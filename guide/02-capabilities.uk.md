# Можливості Claude Cowork

🌐 **Мови**: [English](02-capabilities.md) | [Français](02-capabilities.fr.md) | [Українська 🇺🇦](02-capabilities.uk.md)

Claude Cowork — це потужна комбінація великої мовної моделі (LLM) та можливості виконання дій на комп'ютері. Цей розділ детально описує, що саме вміє робити система.

---

## 🏗️ Основні функції

### 1. Маніпуляції з файлами та папками
Claude може бачити структуру ваших папок і змінювати її.
- **Дії**: Створення, видалення, перейменування, переміщення та копіювання.
- **Впорядкування**: Автоматичне сортування завалених папок (наприклад, `Downloads` або `Desktop`).
- **Синхронізація**: Підтримка структури між різними папками проєкту.

### 2. Читання та аналіз документів
Cowork підтримує більшість офісних форматів:
- **Текстові**: `.txt`, `.md`, `.rtf`.
- **Офісні**: `.docx`, `.xlsx`, `.pptx`, `.csv`.
- **PDF**: Читання тексту, таблиць та метаданих.
- **Зображення**: Аналіз скриншотів, фотографій документів.

### 3. OCR (Розпізнавання тексту)
Claude вміє вилучати текст із зображень та відсканованих PDF.
- **Рахунки**: Вилучення дати, суми, назви постачальника.
- **Чеки**: Оцифрування витрат для бухгалтерської звітності.
- **Рукописний текст**: Висока точність розпізнавання розбірливого почерку.

### 4. Робота з кодом (не тільки для програмістів)
Навіть якщо ви не пишете код, Claude може використовувати його для ваших завдань.
- **Скрипти**: Написання та запуск Python-скриптів для обробки даних в Excel.
- **HTML/CSS**: Створення та перегляд простих веб-сторінок.

## 🔌 Інтеграція через MCP

**Model Context Protocol (MCP)** дозволяє Claude підключатися до ваших улюблених інструментів.

| Сервіс | Що може робити Claude |
| :--- | :--- |
| **Google Drive** | Пошук, читання та редагування файлів у хмарі. |
| **Slack** | Написання повідомлень, читання каналів, підсумовування обговорень. |
| **Notion** | Створення сторінок, оновлення баз даних. |
| **GitHub** | Керування репозиторіями, написання коду, робота з Issue. |

## ⚖️ Обмеження та нюанси

Розуміння меж системи допоможе вам досягти найкращих результатів.

- **Вікно контексту**: Claude має величезну пам'ять (~200,000 токенів), але вона не нескінченна. Якщо ви попросите проаналізувати 500 великих PDF одночасно, система може почати втрачати деталі.
- **Галюцинації**: Хоча Claude дуже точний, він все одно може придумати факт або помилитися в цифрі. Завжди перевіряйте критично важливі дані.
- **Виконання в пісочниці**: Будь-яка команда виконується в ізольованому середовищі. Це безпечно, але означає, що Claude не має доступу до ваших паролів або системних налаштувань Mac без вашого дозволу.

## 🎯 Найкращі випадки використання

1. **Фінанси**: Оцифрування папки з рахунками в одну таблицю Excel.
2. **Маркетинг**: Створення 10 варіантів постів для соцмереж на основі одного прес-релізу.
3. **HR**: Первинний відбір резюме за заданими критеріями.
4. **Адміністрування**: Впорядкування хаосу в папках клієнтів.
