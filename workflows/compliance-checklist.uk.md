# Воркфлоу: Чек-лист Відповідності

> **Орієнтовний час**: 30 хвилин
> **Складність**: Середній
> **Категорія**: 📋 Адміністративні
> **Професії**: Підрядники (будівництво), Торговці, Малий бізнес

🌐 **Мови**: [English](compliance-checklist.en.md) | [Français](compliance-checklist.md) | [Українська 🇺🇦](compliance-checklist.uk.md)

---

## Кейс використання

Вам потрібно переконатися, що ваш бізнес має всі обов'язкові документи та сертифікати для законної діяльності, участі в тендерах або проходження аудитів. Цей воркфлоу допоможе вам:

- Перевірити наявність усіх необхідних документів (страхування, ліцензії, дозволи).
- Підготувати пакети документів для клієнтів або перевірок.
- Відстежувати терміни дії та поновлення.
- Створювати переліки відсутніх документів.

> **Важливо**: Вимоги відрізняються залежно від галузі, локації та типу бізнесу. Цей воркфлоу надає загальні рекомендації — завжди перевіряйте специфічні правила вашого сектора (наприклад, ДБН для будівництва або вимоги Держпродспоживслужби для торгівлі).

## Покрокова інструкція

### Крок 1: Визначте сферу перевірки

```bash
mkdir -p ~/Cowork-Workspace/{input/documents,output/compliance}

# Скопіюйте наявні документи для аналізу
cp ~/Documents/Business/Insurance/*.pdf ~/Cowork-Workspace/input/documents/
cp ~/Documents/Business/Licenses/*.pdf ~/Cowork-Workspace/input/documents/
```

Опишіть свій бізнес для Cowork:

```
Профіль бізнесу для перевірки відповідності:

ІНФОРМАЦІЯ ПРО БІЗНЕС:
- Тип: [ТОВ, ФОП 2/3 група]
- Галузь: [Будівництво, Ритейл, Послуги, Виробництво]
- Локація: [Україна, область]
- Кількість працівників: [Кількість]
- Масштаб: [Локальний, Національний, Експорт]

СПЕЦИФІЧНА ДІЯЛЬНІСТЬ:
- Будівельні/ремонтні роботи? ТАК/НІ
- Робота з продуктами харчування? ТАК/НІ
- Небезпечні матеріали/роботи? ТАК/НІ
- Професійні послуги (консультації)? ТАК/НІ

Згенеруй повний чек-лист відповідності (compliance checklist) на основі цього профілю згідно з законодавством України.
```

### Крок 2: Генерація галузевого чек-листа

Для будівництва (Contractors):

```
Чек-лист відповідності для будівельного бізнесу (Україна):

КАТЕГОРІЯ: ОБОВ'ЯЗКОВЕ СТРАХУВАННЯ ТА ДОЗВОЛИ
☐ Дозвіл на виконання робіт підвищеної небезпеки (Держпраці)
☐ Декларація відповідності матеріально-технічної бази
☐ Страхування цивільної відповідальності перед третіми особами
☐ Страхування будівельно-монтажних ризиків (якщо вимагає замовник)

КАТЕГОРІЯ: ЛІЦЕНЗІЇ ТА РЕЄСТРАЦІЇ
☐ Витяг з ЄДРПОУ (актуальні КВЕДи)
☐ Будівельна ліцензія (якщо об'єкти СС2/СС3)
☐ Реєстрація платника податків (ПДВ/Єдиний податок)
☐ Свідоцтво про реєстрацію ТОВ/ФОП

КАТЕГОРІЯ: СЕРТИФІКАЦІЯ ПЕРСОНАЛУ
☐ Сертифікат інженера-проектувальника/архітектора (якщо є)
☐ Посвідчення з охорони праці та електробезпеки
☐ Проходження медичних оглядів персоналом

Збережи як: ~/Cowork-Workspace/output/compliance/checklist-construction.xlsx
```

Для торгівлі (Retail):

```
Чек-лист відповідності для роздрібної торгівлі (Україна):

КАТЕГОРІЯ: РЕЄСТРАЦІЯ ТА ПОДАТКИ
☐ Витяг з ЄДР (торгові КВЕДи)
☐ Реєстрація РРО/ПРРО (фіскалізація чеків)
☐ Договір оренди або свідоцтво на право власності приміщення
☐ Куточок споживача (правила торгівлі, книга відгуків - за бажанням)

КАТЕГОРІЯ: ДОЗВОЛИ ТА БЕЗПЕКА
☐ Декларація пожежної безпеки (ДСНС)
☐ Дозвіл на розміщення зовнішньої реклами/вивіски
☐ Експлуатаційний дозвіл (якщо робота з продуктами тваринного походження)

КАТЕГОРІЯ: САНІТАРНІ ВИМОГИ (якщо продукти/кафе)
☐ Санітарні книжки працівників
☐ Впровадження системи HACCP
☐ Договір на дератизацію та вивіз сміття

Збережи як: ~/Cowork-Workspace/output/compliance/checklist-retail.xlsx
```

### Крок 3: Перевірка наявних документів

```
Проскануй папку з документами: ~/Cowork-Workspace/input/documents/

Для кожного знайденого документа:
1. Витягни: Тип документа, дату закінчення, орган, що видав.
2. Зістав із пунктами чек-листа.
3. Познач статус: ✅ Валідний | ⚠️ Закінчується (<30 днів) | ❌ Прострочений | ❓ Відсутній

Згенеруй звіт про стан відповідності (Compliance Status Report).
Збережи у: ~/Cowork-Workspace/output/compliance/status-report-[date].pdf
```

### Крок 4: Створення плану дій

```
На основі звіту створи пріоритетний план дій:

ПРІОРИТЕТ 1 - КРИТИЧНО (Виконати протягом 7 днів):
- Поновити прострочені дозволи.
- Оформити обов'язкове страхування.

ПРІОРИТЕТ 2 - ТЕРМІНОВО (Виконати протягом 30 днів):
- Продовжити ліцензії, що закінчуються.
- Провести навчання персоналу з охорони праці.

Збережи як: ~/Cowork-Workspace/output/compliance/action-plan-[date].xlsx
Додай колонки для відповідальних осіб та дедлайнів.
```

### Крок 5: Підготовка пакета для клієнта (Тендер)

```
Клієнт/Замовник вимагає пакет документів для проєкту [Назва проєкту]

Сформуй пакет відповідності:
1. ТИТУЛЬНА СТОРІНКА: Назва компанії, дата, перелік документів.
2. РОЗДІЛИ:
   - Реєстраційні документи (Витяг, ПДВ).
   - Ліцензії та дозволи.
   - Страхові поліси.
   - Сертифікати якості/кваліфікації.

Формат: Єдиний PDF-файл зі змістом.
Збережи у: ~/Cowork-Workspace/output/compliance/client-package-[client-name].pdf
```

## Найкращі практики

1. **Календарні нагадування** — Встановлюйте сповіщення за 60 днів до закінчення терміну дії важливих документів.
2. **Центральне сховище** — Зберігайте всі оригінали та цифрові копії в одному захищеному місці.
3. **Щорічний аудит** — Проводьте повну перевірку відповідності щороку (найкраще — у січні).
4. **Відстеження змін** — Слідкуйте за оновленнями законодавства у вашій галузі (наприклад, через ресурси Ліга:Закон або офіційні сайти міністерств).
5. **Професійна консультація** — У разі складних питань (наприклад, отримання ліцензії на СС3) звертайтеся до профільних юристів.
