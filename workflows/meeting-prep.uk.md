# Воркфлоу: Підготовка до зустрічі

🌐 **Мови**: [English](meeting-prep.en.md) | [Français](meeting-prep.md) | [Українська 🇺🇦](meeting-prep.uk.md)

> **Орієнтовний час**: 20 хвилин
> **Складність**: Середня
> **Категорія**: 🗂️ Організація
> **Професії**: Бізнес-лідери, Фрілансери

---

## Кейс використання

У вас запланована зустріч, і вам потрібно:

- Провести дослідження компанії та людей, з якими ви зустрічаєтеся
- Зібрати відповідні довідкові матеріали
- Створити бриф-документ
- Підготувати тези для виступу та запитання

---

## Попередні вимоги

- [ ] Cowork увімкнено з доступом до Chrome (для веб-досліджень)
- [ ] Наявні документи щодо зустрічі/компанії
- [ ] Чітке розуміння мети зустрічі

---

## Покрокова інструкція

### Крок 1: Налаштування робочого простору

```bash
# Створіть робочий простір для підготовки
mkdir -p ~/Cowork-Workspace/input/meeting-[name]
mkdir -p ~/Cowork-Workspace/output

# Скопіюйте наявні матеріали
cp ~/Documents/relevant-files/* ~/Cowork-Workspace/input/meeting-[name]/
```

### Крок 2: Визначення контексту зустрічі

Розкажіть Cowork про зустріч:

```
У мене зустріч із [Компанія/Особа] [Дата] о [Час].

Мета зустрічі: [опишіть]
Моя роль: [ваша посада/компанія]
Їхня роль: [з ким ви зустрічаєтеся]
Чого я хочу досягти: [ваші цілі]

Цей контекст потрібен для бриф-документа, який ти створиш.
```

### Крок 3: Етап дослідження

**Дослідження компанії:**
```
Проведи дослідження [Назва компанії] та створи бриф.

Включи:
1. Огляд компанії (чим займаються, розмір, штаб-квартира, ключові продукти).
2. Останні новини (за останні 3-6 місяців).
3. Керівництво (CEO та ключові топ-менеджери).
4. Позиція на ринку (основні конкуренти, поточний стан).

Зберегти у: ~/Cowork-Workspace/output/company-research.md
```

**Дослідження особи:**
```
Проведи дослідження [Ім'я особи] у [Компанія].

Знайди:
- Поточна посада та обов'язки
- Кар'єрний шлях та освіта
- Публічні виступи/публікації
- Ключові тези з LinkedIn
- Спільні зв'язки або інтереси

Зберегти у: ~/Cowork-Workspace/output/person-profile.md
```

### Крок 4: Створення бриф-документа

```
Створи бриф-документ для зустрічі, використовуючи:
- Зібрані дослідження
- Матеріали з ~/Cowork-Workspace/input/meeting-[name]/

Структура:
1. Огляд зустрічі (дата, час, місце, учасники, мета).
2. Довідка про компанію (ключові факти та останні події).
3. Профілі учасників (коротко про кожну людину).
4. Наша історія (попередні взаємодії, якщо надані матеріали).
5. Тези для виступу (5-7 тем та ключові повідомлення).
6. Запитання (5-7 стратегічних та уточнюючих запитань).
7. Потенційні заперечення (можливі побоювання та варіанти відповідей).
8. Цілі зустрічі (що вважатиметься успіхом).

Формат: документ Word
Зберегти у: ~/Cowork-Workspace/output/meeting-brief.docx
```

### Крок 5: Створення картки швидкого доступу

```
На основі брифу створи односторінкову картку швидкого доступу.

Включи лише:
- Ключові факти про компанію/особу
- Основні тези (маркований список)
- Критичні запитання
- Імена, які слід пам'ятати

Формат: для друку та використання під час зустрічі
Зберегти у: ~/Cowork-Workspace/output/quick-reference.docx
```

## Варіації за типами зустрічей

### Зустріч із продажу / з клієнтом
Фокус на бізнес-викликах клієнта, нашій відповідності його потребам, стратегії продажів та наступних кроках.

### Співбесіда
Фокус на культурі компанії, аналізі ролі, вашій відповідності вимогам (у форматі STAR) та запитаннях до роботодавця.

### Зустріч із партнером / постачальником
Фокус на оцінці постачальника, критеріях вибору, матриці запитань (технічних, цінових) та порівнянні з альтернативами.

### Зустріч із керівництвом / радою директорів
Фокус на порядку денному, необхідних рішеннях, рекомендаціях з обґрунтуванням та карті стейкхолдерів.

## Усунення несправностей

### Мало інформації про компанію
Для малих або приватних компаній зосередьтеся на їхньому сайті, сторінці в LinkedIn та загальному контексті галузі. Позначте прогалини в брифі, щоб дослідити їх під час зустрічі.

### Застаріла інформація
Просіть Cowork перевіряти актуальність даних та позначати те, що може потребувати підтвердження.

### Спільне ім'я
Уточнюйте пошук за посадою, містом або надавайте посилання на профіль у LinkedIn.

---

## Найкращі практики

1. **Починайте заздалегідь** — веб-дослідження потребує часу.
2. **Фокусуйтеся на релевантності** — не всі факти про компанію однаково важливі.
3. **Готуйте запитання** — це демонструє залученість та спрямовує розмову.
4. **Знайте свої цілі** — чіткі цілі допомагають побудувати правильний бриф.
5. **Оновлюйте та використовуйте повторно** — брифи для регулярних зустрічей можна легко актуалізувати.
