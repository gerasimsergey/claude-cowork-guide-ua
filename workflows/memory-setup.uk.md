# Налаштування постійної пам'яті

🌐 **Мови**: [English](memory-setup.en.md) | [Français](memory-setup.md) | [Українська 🇺🇦](memory-setup.uk.md)

> **Категорія**: Організація | **Складність**: Початкова | **Час налаштування**: 15 хв

---

## Мета

Перестаньте повторюватися. Без належного налаштування вам доводиться щоразу заново пояснювати свої вподобання, дані про клієнтів та робочі звички на початку кожної сесії Cowork. Завдяки файлу `memory.md` та Desktop Commander, Cowork знатиме все це автоматично.

## Попередні вимоги

- Активний Cowork (передплата Pro або Max)
- Встановлений та оновлений Claude Desktop
- 15 хвилин для початкового налаштування

---

## Кроки

### Крок 1: Встановлення Desktop Commander

Desktop Commander — це офіційне розширення, яке забезпечує роботу постійної пам'яті.

1. У Claude Desktop відкрийте вкладку **Customize** (головна навігація).
2. Знайдіть **Desktop Commander** у списку розширень.
3. Натисніть **Install**.
4. Перезапустіть Claude Desktop за запитом.

> Детальніше див. у [Початок роботи, крок 8](../guide/01-getting-started.md#крок-8-встановіть-desktop-commander-рекомендовано).

### Крок 2: Створення memory.md

1. У Finder перейдіть до папки `~/Cowork-Workspace/`.
2. Створіть новий текстовий файл із назвою `memory.md`.
3. Відкрийте його за допомогою TextEdit або будь-якого текстового редактора.

### Крок 3: Заповнення вашої пам'яті

Скопіюйте шаблон для вашого типу бізнесу (див. наступний розділ) та адаптуйте його під свою ситуацію.

Почніть із простого: 5-10 рядків достатньо для старту. Ви будете наповнювати його з часом.

### Крок 4: Перевірка роботи

Відкрийте нову сесію Cowork і введіть:

```
Прочитай ~/Cowork-Workspace/memory.md і резюмуй, що ти знаєш про мій бізнес.
```

Якщо Cowork точно описує ваш контекст — налаштування виконано правильно.

### Крок 5: Використання в кожній сесії

Відтепер починайте кожен діалог із фрази:

```
Спершу прочитай ~/Cowork-Workspace/memory.md. Потім [ваш запит]
```

## Шаблони за типами бізнесу

### Майстер / Фахівець (сантехнік, електрик, будівельник)

```markdown
# Пам'ять Cowork — [Ім'я Прізвище], [Професія]

## Мій бізнес
- Професія: [ваша професія]
- Територія обслуговування: [міста/регіони]
- Команда: [працюю сам / X співробітників]

## Постійні клієнти
- Родина Коваль (Львів): квартира 70-х років, завжди потрібні деталізовані кошториси, оплата протягом 30 днів.
- Агентство нерухомості "Мій Дім": роботи між орендарями, виставлення рахунків наприкінці місяця.

## Стандарти кошторисів та рахунків
- Формат кошторису: робота та матеріали вказуються окремо, ПДВ відображається чітко.
- Умови оплати: 30% аванс, залишок після завершення.
- Термін дії кошторису: 3 місяці.

## Чого я ніколи не роблю
- Не обіцяю дату без попередньої перевірки графіка.
- Не надсилаю рахунок без звірки з журналом робіт.
- Не використовую технічний жаргон у документах для клієнтів.
```

### Роздрібна торгівля (магазин, крамниця, склад)

```markdown
# Пам'ять Cowork — [Назва магазину]

## Мій бізнес
- Тип: [опис]
- Площа: [кв.м] | Персонал: [кількість]
- Основні клієнти: [профіль]

## Ключові постачальники
- [Постачальник А]: термін виконання 3 тижні, мін. замовлення $500, контакт [ім'я].
- [Постачальник Б]: доставка 48 год, без мінімального замовлення.

## Топ-товари (завжди в наявності)
- [Товар 1]: арт. [X], поріг замовлення [Y].
```

## Універсальний шаблон (якщо не знаєте, з чого почати)

```markdown
# Моя пам'ять Cowork

## Мій бізнес
- Сектор: [ваш сектор]
- Розмір: [працюю сам / малий бізнес / команда з X осіб]

## Постійний контекст
- [Інформація 1, яку ви часто повторюєте]
- [Інформація 2]

## Мої вподобання
- Тон: [офіційний / неформальний]
- Улюблені формати: [Word / Excel / PDF]
- Чого я ніколи не роблю: [список]
```

## Поради з використання

**Почніть з 5-10 рядків.** Короткий `memory.md` кращий за складний файл, який ви ніколи не оновлюєте. Додавайте інформацію, коли помічаєте, що повторюєте щось сесію за сесією.

**Оновлюйте регулярно.** Якщо вподобання клієнта змінилися, постачальник переїхав або умови оплати еволюціонували — оновіть файл. Застаріла інформація може призвести до помилок.

**Тестуйте щомісяця.** Раз на місяць просіть Cowork резюмувати, що він знає про ваш бізнес. Це підтвердить, що файл читається, і нагадає вам про його зміст.
