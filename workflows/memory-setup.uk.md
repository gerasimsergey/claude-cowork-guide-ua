# Воркфлоу: Налаштування Пам'яті Бізнесу

> **Орієнтовний час**: 15 хвилин
> **Складність**: Початківець
> **Категорія**: 🗂️ Організація

🌐 **Мови**: [English](memory-setup.en.md) | [Français](memory-setup.md) | [Українська 🇺🇦](memory-setup.uk.md)

---

## Мета

Припиніть повторюватися. Без налаштування пам'яті ви змушені пояснювати свої вподобання, дані про клієнтів та робочі звички на початку кожної сесії Cowork. За допомогою файлу `memory.md` Клод знатиме все це автоматично.

---

## Кроки налаштування

### Крок 1: Створення файлу memory.md

1. У Finder перейдіть до папки `~/Cowork-Workspace/`.
2. Створіть новий текстовий файл з назвою `memory.md`.
3. Відкрийте його для редагування.

### Крок 2: Наповнення пам'яті за шаблоном

Скопіюйте шаблон для вашого типу бізнесу (див. нижче) та адаптуйте його під себе. Почніть з малого (5-10 рядків), ви будете доповнювати його з часом.

### Крок 3: Перевірка роботи

Відкрийте нову сесію Cowork та запитайте:
```
Прочитай ~/Cowork-Workspace/memory.md та коротко розкажи, що ти знаєш про мій бізнес.
```

### Крок 4: Використання в роботі

Тепер починайте кожну нову розмову (якщо це новий чат) фразою:
```
Спочатку прочитай ~/Cowork-Workspace/memory.md. А потім [ваш запит].
```

## Шаблони за видами діяльності

### Для Майстрів та Підрядників (Електрик, Сантехнік, Будівельник)

```markdown
# Пам'ять Cowork — [Ваше Ім'я], [Ваша Спеціальність]

## Мій Бізнес
- Спеціалізація: [Напр: Електромонтажні роботи]
- Регіон роботи: [Місто, область]
- Форма власності: [ФОП 2-га група / ТОВ]

## Постійні Клієнти
- ЖК "Центральний" (Одеса): Виконуємо обслуговування щитових, оплата за актами раз на місяць.
- БЦ "Океан": Потрібні детальні звіти з фотофіксацією.

## Стандарти Кошторисів та Рахунків
- Валюта: UAH (Гривня).
- Умови оплати: Аванс 50% на матеріали, залишок по завершенню.
- Термін дії пропозиції: 14 днів (через нестабільність цін на метал).

## Мої Правила
- Ніколи не обіцяю точну дату без перевірки графіка.
- Завжди вказую ПДВ окремим рядком (якщо актуально).
- Використовую тільки професійну термінологію в техзавданнях.

## Постачальники
- Головний постачальник: [Назва, контакт] (маємо знижку 10%).
```

### Для Торгівлі та Магазинів

```markdown
# Пам'ять Cowork — [Назва Магазину]

## Про Бізнес
- Тип: [Роздрібна торгівля будматеріалами]
- Команда: [Кількість співробітників]
- Цільова аудиторія: [Професійні майстри / Приватні особи]

## Ключові Постачальники
- [Постачальник А]: Доставка 3 дні, мін. замовлення 5000 грн.
- [Постачальник Б]: Самовивіз зі складу в [Місто].

## Пріоритетні Товари
- Цемент М-500: артикул [X], тримати залишок не менше 20 мішків.

## Формати Документів
- Замовлення постачальнику: Таблиця Excel з ПДВ.
- Оголошення для клієнтів: Дружній тон, акцент на якості.

## Мої Правила
- Не роблю замовлення, поки не перевірю залишки на складі.
- Не обіцяю знижку більше 5% без погодження.
```

## Поради з використання

1. **Почніть з короткого** — Короткий, але точний файл `memory.md` кращий за складний документ, який ви не оновлюєте.
2. **Оновлюйте регулярно** — Змінився постачальник чи умови оплати? Відразу внесіть правки у файл.
3. **Використовуйте для стилю** — Напишіть у пам'яті: "Я звертаюся до клієнтів на Ви, але в команді ми спілкуємося неформально". Клод підхопить цей стиль у всіх листах.
4. **Тестуйте раз на місяць** — Просіть Клода переказати вміст пам'яті. Це нагадає вам, що там написано, і допоможе знайти застарілу інформацію.
5. **Безпека** — Не зберігайте в пам'яті паролі або дуже чутливі дані. Достатньо загальної бізнес-інформації.
