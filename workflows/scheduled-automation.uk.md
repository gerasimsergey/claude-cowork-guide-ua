# Воркфлоу: Запланована Автоматизація

> **Орієнтовний час налаштування**: 30 хвилин
> **Складність**: Середня
> **Категорія**: 🗂️ Організація

🌐 **Мови**: [English](scheduled-automation.en.md) | [Français](scheduled-automation.md) | [Українська 🇺🇦](scheduled-automation.uk.md)

---

## Мета

Автоматизувати повторювані завдання (звіти, зведення даних, тижневі огляди) так, щоб вони виконувалися без вашої ручної участі. Налаштуйте один раз — отримуйте результати щодня, щотижня або щомісяця.

---

## Попередні вимоги

- Активна підписка (Pro або Max).
- Claude Desktop встановлено та налаштовано.
- Папки вхідних даних розділені за частотою: `daily/`, `weekly/`, `monthly/`.

```bash
mkdir -p ~/Cowork-Workspace/input/{daily,weekly,monthly}
mkdir -p ~/Cowork-Workspace/output/{reports,dashboards}
```

---

## 4 Основні шаблони автоматизації

### Шаблон 1: Ранковий брифінг (Daily Morning Brief)

**Коли**: Щодня о 09:00.
**Результат**: Огляд завдань на сьогодні на одну сторінку.

**Промпт для налаштування**:
```
КОНТЕКСТ: Файли, додані вчора в ~/Cowork-Workspace/input/daily/
ЗАВДАННЯ: Проаналізуй нові документи та визнач дії на сьогодні.
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/reports/brief-[дата].md
СТРУКТУРА: "Терміново", "Сьогодні", "Для ознайомлення".
ОБМЕЖЕННЯ: Максимум 1 сторінка. Тільки головне. Виділи протерміновані справи.
```

---

### Шаблон 2: Щотижневий звіт (Weekly Sales Compilation)

**Коли**: Щопонеділка о 08:00.
**Результат**: Повне зведення результатів минулого тижня.

**Промпт для налаштування**:
```
КОНТЕКСТ: Усі файли в ~/Cowork-Workspace/input/weekly/ за останні 7 днів.
ЗАВДАННЯ: Склади тижневе зведення — досягнення, ключові цифри, незавершені справи, необхідні фоллоу-апи.
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/reports/weekly-summary-[дата].docx
ОБМЕЖЕННЯ: Професійний стиль, до 2 сторінок. Жирним шрифтом виділи критичні моменти.
```

---

### Шаблон 3: Підсумок п'ятниці (Friday Wrap-Up)

**Коли**: Щоп'ятниці о 17:30.
**Результат**: План передачі справ на наступний понеділок.

**Промпт для налаштування**:
```
КОНТЕКСТ: Файли, змінені або створені цього тижня в ~/Cowork-Workspace/
ЗАВДАННЯ: Створи підсумок тижня — що зроблено, що в процесі, що чекає на понеділок.
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/reports/recap-[дата].md
ОБМЕЖЕННЯ: Короткі речення, список пріоритетів на ранок понеділка.
```

---

### Шаблон 4: Місячний дашборд (Monthly Dashboard)

**Коли**: 1-го числа кожного місяця о 07:00.
**Результат**: Огляд місяця з порівнянням показників.

**Промпт для налаштування**:
```
КОНТЕКСТ: Файли в ~/Cowork-Workspace/input/monthly/ за минулий місяць.
ЗАВДАННЯ: Створи місячний дашборд — ключові метрики, тренди, важливі події, прогноз.
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/dashboards/dashboard-[місяць-рік].xlsx
ОБМЕЖЕННЯ: Порівняй із попереднім місяцем. Червоним виділи показники, що нижче плану.
```

## Як налаштувати в Claude Desktop

1. Відкрийте **Claude Desktop → Ліва бічна панель → Розділ Scheduled**.
2. Натисніть **New Task**.
3. Оберіть тип: **Recurring** (Повторюване).
4. Вставте промпт (оберіть один із шаблонів вище).
5. Налаштуйте час та періодичність (напр. Weekly, Monday, 08:00).
6. Дайте назву (напр. "Звіт за тиждень").
7. Активуйте.

## Важливі примітки

1. **Комп'ютер має бути увімкненим** — Для виконання завдання Claude Desktop має бути відкритим, а комп'ютер не повинен бути в режимі сну. Якщо комп'ютер спав, завдання виконається відразу після його пробудження.
2. **Перевіряйте перші випуски** — Перші 2-3 рази перевіряйте результат автоматизації вручну, щоб переконатися, що Claude правильно розуміє структуру ваших даних.
3. **Оптимізація промптів** — Після першого запуску Cowork автоматично запропонує покращити ваш промпт для кращої точності. Ви можете погодитися або залишити свій.
4. **Імена файлів** — Використовуйте теги дати в назвах файлів (напр. `[дата]`), щоб вони не перезаписували один одного.
