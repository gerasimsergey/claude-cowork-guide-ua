# Планова автоматизація

🌐 **Мови**: [English](scheduled-automation.en.md) | [Français](scheduled-automation.md) | [Українська 🇺🇦](scheduled-automation.uk.md)

> **Категорія**: Організація | **Складність**: Середня | **Час налаштування**: 30 хв

---

## Мета

Автоматизуйте повторювані завдання, звіти, компіляції та тижневі огляди, щоб вони генерувалися без ручних зусиль. Налаштуйте один раз — отримуйте результати щодня, щотижня або щомісяця.

## Попередні вимоги

- Активний Cowork (передплата Pro або Max)
- Ознайомлення з фреймворком CTOC ([Початок роботи, крок 5](../guide/01-getting-started.md#крок-5-фреймворк-ctoc))
- Організовані вхідні папки: розділені за частотою (daily/, weekly/, monthly/)

## Підготовка папок

Перед налаштуванням запланованих завдань організуйте робочий простір:

```bash
mkdir -p ~/Cowork-Workspace/input/{daily,weekly,monthly}
mkdir -p ~/Cowork-Workspace/output/{reports,dashboards}
```

Розміщуйте файли у відповідних підпапках залежно від того, як часто вони потребують обробки.

## 4 основні шаблони

### Шаблон 1: Щоденний ранковий бриф (Morning Brief)

**Коли**: Щодня о 09:00
**Час налаштування**: 10 хвилин
**Результат**: Односторінковий огляд того, що потребує вашої уваги сьогодні.

**Промпт для налаштування**:
```
КОНТЕКСТ: Файли, додані вчора до ~/Cowork-Workspace/input/daily/
ЗАВДАННЯ: Проаналізуй нові документи та визнач дії на сьогодні
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/reports/brief-[дата].md
Структура: 3 розділи — "Термінові дії", "Зробити сьогодні", "До відома"
ОБМЕЖЕННЯ: Максимум 1 сторінка. Тільки марковані списки. Познач термінові справи або прострочені завдання.
```

---

### Шаблон 2: Тижневий звіт із продажів

**Коли**: Щопонеділка о 08:00
**Час налаштування**: 15 хвилин
**Результат**: Повний підсумок роботи за минулий тиждень.

**Промпт для налаштування**:
```
КОНТЕКСТ: Усі файли в ~/Cowork-Workspace/input/weekly/ за останні 7 днів
ЗАВДАННЯ: Склади тижневий підсумок — досягнення, ключові цифри, незавершені справи, необхідні подальші кроки
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/reports/weekly-summary-[дата].docx
Структура: Резюме (1 абзац), Досягнення, Цифри (якщо є), Очікування, Дії на понеділок
ОБМЕЖЕННЯ: Діловий формат, макс. 2 сторінки. Виділи жирним критичні моменти.
```

---

### Шаблон 3: Підсумок п'ятниці (Friday Wrap-Up)

**Коли**: Щоп'ятниці о 17:30
**Час налаштування**: 10 хвилин
**Результат**: Документ для передачі справ на наступний понеділок.

**Промпт для налаштування**:
```
КОНТЕКСТ: Файли, змінені або створені цього тижня в ~/Cowork-Workspace/
ЗАВДАННЯ: Створи підсумок кінця тижня — що зроблено, що в процесі, що чекає до понеділка
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/reports/recap-[дата].md
Структура: Зроблено цього тижня, У процесі (статус), Пріоритети на ранок понеділка (рейтинг)
ОБМЕЖЕННЯ: Орієнтація на дію. Короткі речення. Максимум одна сторінка.
```

---

### Шаблон 4: Місячний дашборд

**Коли**: 1-го числа кожного місяця о 07:00
**Час налаштування**: 20 хвилин
**Результат**: Місячний огляд із порівнянням із попереднім місяцем.

**Промпт для налаштування**:
```
КОНТЕКСТ: Файли в ~/Cowork-Workspace/input/monthly/ за минулий місяць
ЗАВДАННЯ: Створи місячний дашборд — ключові метрики, тренди, помітні події, прогноз
РЕЗУЛЬТАТ: ~/Cowork-Workspace/output/dashboards/dashboard-[місяць-рік].xlsx
Структура: Лист "Summary" (огляд для керівництва) + Лист "Detail" (сирі дані)
ОБМЕЖЕННЯ: Використовуй синтаксис формул з розділювачами-комами. Порівнюй із попереднім місяцем, де це дозволяють дані. Виділяй червоним усе, що нижче цільових показників.
```

## Як налаштувати заплановане завдання в Claude Desktop

### Два типи завдань

**Регулярні завдання (Recurring tasks)** — запускаються автоматично з заданою періодичністю: щогодини, щодня, щотижня, тільки в робочі дні або за власним графіком. Після першого запуску Cowork автоматично переписує ваш промпт, щоб оптимізувати його на основі отриманого досвіду.

**Завдання за запитом (On-demand tasks)** — запускаються один раз при натисканні "Запустити зараз". Корисно для нерегулярних або разових операцій без створення постійного графіка.

### Кроки налаштування

1. Відкрийте **Claude Desktop → ліва бічна панель → розділ Scheduled**.
2. Натисніть **New Task**.
3. Оберіть тип (регулярне або за запитом).
4. Вставте ваш промпт.
5. Для регулярних: встановіть періодичність та час.
6. Назвіть завдання (напр., "Ранковий бриф", "Звіт понеділка").
7. Активуйте.

## Важливі замітки

- **Надійність**: Заплановані завдання знаходяться на етапі дослідницької прев'ю-версії. Перші запуски можуть бути недосконалими. Перевіряйте результати вручну.
- **Додаток ВІДКРИТИЙ і пристрій НЕ СПИТЬ**: Cowork вимагає, щоб Claude Desktop був відкритим, а комп'ютер — увімкненим (не в режимі сну) у запланований час. Якщо пристрій спить або додаток закритий, завдання **пропускається** і запускається автоматично при пробудженні.
- **Автоматична оптимізація промптів**: Після першого запуску регулярного завдання Cowork аналізує результат і переписує ваш промпт для його покращення. Перевіряйте оновлений промпт у бічній панелі.

---

## Поради з оптимізації

- **Почніть з одного шаблону**: Запустіть одне завдання на 2 тижні, перш ніж додавати інші.
- **Перевіряйте перші результати**: Перші 3-4 запуски потребують ретельного перегляду.
- **Використовуйте дати в назвах файлів**: Це полегшить пошук в архівах пізніше.
