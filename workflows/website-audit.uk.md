## Кейс використання

Ваш сайт — це перше враження потенційного клієнта. Цей воркфлоу допоможе вам:

- Виявити критичні помилки, які заважають клієнтам зателефонувати вам.
- Перевірити доступність контактної інформації.
- Оцінити швидкість та зручність сайту для мобільних пристроїв.
- Порівняти ваш сайт із конкурентами на місцевому ринку.

> ⚠️ **Обмеження**: Claude може переглядати ваш сайт через вбудований браузер, але не може повноцінно протестувати мобільну версію фізично. Аналіз базується на візуальній структурі та коді.

## Покрокова інструкція

### Крок 1: Підготовка до аудиту

```bash
mkdir -p ~/Cowork-Workspace/output/website-audit
```

### Крок 2: Комплексний аудит сайту

Надайте Claude посилання та запустіть перевірку:

```
Відкрий мій сайт: [Ваш URL]

Проведи повний аудит за наступними критеріями:

1. КОНТАКТНА ІНФОРМАЦІЯ (Критично)
- Чи видно номер телефону в перші 3 секунди?
- Чи є посилання на Telegram/Viber?
- Чи вказана фізична адреса та графік роботи?
- Чи працює кнопка "Замовити дзвінок"?

2. МОВНА ЛОКАЛІЗАЦІЯ (Закон України)
- Чи є українська версія сайту за замовчуванням?
- Чи всі елементи (кнопки, форми) перекладені коректно?

3. ДОВІРА ТА СОЦІАЛЬНІ ДОКАЗИ
- Чи є блок з відгуками?
- Чи є фото реальних робіт/проєктів (не зі стоків)?
- Чи вказано досвід роботи та сертифікати?

4. ТЕХНІЧНИЙ СТАН
- Чи працює захищений протокол HTTPS?
- Чи швидко завантажуються фото?
- Чи немає битих посилань?

Згенеруй підсумок: 3 КРИТИЧНІ ПОМИЛКИ та 3 СИЛЬНІ СТОРОНИ.
Збережи у: ~/Cowork-Workspace/output/website-audit/audit-report.txt
```

### Крок 3: Створення плану дій

```
На основі аудиту склади пріоритетний план виправлень:

- ТЕРМІНОВО (Цього тижня): Виправлення номерів телефонів, форм зворотного зв'язку.
- ВАЖЛИВО (Протягом місяця): Оновлення фото робіт, додавання відгуків.
- ОПТИМІЗАЦІЯ: Налаштування SEO-заголовків для Google.

Збережи у: ~/Cowork-Workspace/output/website-audit/action-plan.txt
```

### Крок 4: Порівняння з конкурентами

```
Порівняй мій сайт [Ваш URL] із конкурентом [URL конкурента]:
- Хто пропонує послугу чіткіше?
- У кого швидше знайти ціну?
- Яку фішку конкурента варто запозичити?
```

### Крок 5: Базовий SEO-чекап

```
Перевір мета-теги головної сторінки:
- Title: Чи містить він назву послуги та місто (напр. "Електрик Львів")?
- Description: Чи спонукає він клієнта клікнути в результатах пошуку?
- Чи є заголовок H1 на сторінці?
```

## Найкращі практики

1. **Мобільна першість (Mobile First)** — 70-80% ваших клієнтів дивляться сайт з телефона. Перевірте, чи не перекриває кнопка месенджера основний текст.
2. **Українська мова** — Згідно із законодавством, українська версія має завантажуватися першою. Це також важливо для локального SEO.
3. **Клікубельні телефони** — Переконайтеся, що при натисканні на номер телефону на сайті телефон клієнта відразу пропонує здійснити виклик.
4. **Реальні обличчя** — Фото вашої команди або вас за роботою викликають у 5 разів більше довіри, ніж ідеальні картинки з інтернету.
5. **Заклик до дії (CTA)** — На кожній сторінці має бути чітка кнопка: "Отримати консультацію", "Замовити розрахунок" або "Записатися".
