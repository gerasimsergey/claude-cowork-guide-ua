## Кейс використання

Ви отримуєте відгуки в Google Maps, Facebook або напряму в месенджерах. Цей воркфлоу допоможе вам:

- Професійно та щиро дякувати за позитивні відгуки.
- Елегантно обробляти негативні коментарі без конфліктів.
- Перетворювати критику на можливість покращити бізнес.
- Створювати публічний образ компанії, яка дбає про клієнтів.

> ⚠️ **Репутація**: 89% споживачів читають відповіді на відгуки перед покупкою. Наявність відповідей підвищує довіру до бренду на 35%.

## Покрокова інструкція

### Крок 1: Аналіз відгуку

Надайте Claude текст відгуку для аналізу:

```
Проаналізуй цей відгук та запропонуй стратегію відповіді:

ВІДГУК:
"[Вставте текст відгуку]"

КОНТЕКСТ:
Бізнес: [Наприклад: Сантехнічні послуги]
Цінності: [Наприклад: Швидкість, Чистота, Чесна ціна]

ЗАПИТ:
1. Визнач тональність (Позитивна/Нейтральна/Негативна).
2. Виділи ключові претензії або подяки.
3. Оціни ризики для репутації.
```

### Крок 2: Генерація адаптованої відповіді

```
Згенеруй професійну відповідь на основі аналізу:

ЯКЩО ПОЗИТИВНИЙ (4-5⭐):
- Тон: Теплий, вдячний.
- Додай: Подяку по імені + згадку деталі (наприклад, "раді, що бойлер працює добре") + запрошення звертатися знову.

ЯКЩО НЕГАТИВНИЙ (1-2⭐):
- Тон: Емпатичний, відповідальний (без виправдань).
- Додай: Визнання проблеми + коротке пояснення (якщо доречно) + конкретне рішення + заклик перейти в приватні повідомлення.

ОБМЕЖЕННЯ:
❌ Уникай шаблонних фраз "Дякуємо за ваш відгук".
❌ Не звинувачуй клієнта ("Ви самі не так зробили").
❌ Не обіцяй неможливого.

Згенеруй 2 варіанти: формальний та більш дружній.
```

### Крок 3: Перевірка перед публікацією

```
Перевір згенеровану відповідь за чек-листом:
- Чи використано ім'я клієнта?
- Чи немає граматичних помилок?
- Чи вказано контактні дані для вирішення проблеми (у разі негативу)?
- Чи виглядає відповідь "людяною"?
```

### Крок 4: Робота з фейковими відгуками

Якщо ви підозрюєте, що відгук залишив конкурент або бот:

```
Склади ввічливу відповідь-запит для ідентифікації:
"Вітаємо! Ми не знайшли запису про ваші роботи в нашій базі за вказаним іменем. Будь ласка, вкажіть номер замовлення або зателефонуйте нам за номером [Телефон], щоб ми могли розібратися в ситуації."
```

### Крок 5: Збір статистики

```
На основі останніх 10 відгуків склади звіт:
- Яка середня оцінка?
- На що скаржаться найчастіше?
- Що клієнти хвалять найбільше?
- Рекомендація: Що змінити в роботі на основі фідбеку?
```

## Найкращі практики

1. **Швидкість має значення** — Відповідайте протягом 24 годин. Це показує, що ви жива компанія.
2. **Публічність — це вітрина** — Пам'ятайте, що вашу відповідь на негатив читатиме не тільки скаржник, а й сотні потенційних клієнтів. Будьте взірцем виваженості.
3. **Персоналізація** — Навіть якщо ви використовуєте шаблони, додайте одну унікальну деталь із відгуку клієнта.
4. **Визнання помилок** — Якщо ви справді помилилися (запізнилися, неякісно зробили) — визнайте це чесно. Це викликає повагу більше, ніж виправдання.
5. **Заклик до дії в приваті** — Ніколи не ведіть довгі суперечки публічно. Напишіть: "Ми хочемо все виправити, напишіть нам у Telegram або зателефонуйте прямо зараз".
