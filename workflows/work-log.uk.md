# Журнал робіт (Work Log)

🌐 **Мови**: [English](work-log.en.md) | [Français](work-log.md) | [Українська 🇺🇦](work-log.uk.md)

Цей воркфлоу допомагає структурувати ваш робочий день, перетворюючи розрізнені нотатки та чати на професійний щоденний або щотижневий звіт про виконану роботу.

---

## 📋 Контекст

Протягом дня ви виконували різні завдання, писали в Slack, робили нотатки в телефоні або блокноті. Тепер вам потрібно звести все це в один зрозумілий звіт для себе, клієнта або керівника.

## 🛠️ Підготовка

1. Зберіть ваші нотатки, скриншоти виконаних робіт або лог чату в один файл (`daily_notes.txt`).
2. Надайте Claude доступ до цього файлу.

---

## 📝 Промпт (Запит)

Скопіюйте цей запит у Claude:

```text
CONTEXT: I need to create a professional daily work report for my client.
TASK: Analyze my rough notes in 'today_work.txt'. 
- Extract all completed tasks.
- Estimate the time spent on each task (if not specified, provide a logical estimate).
- Group tasks by project or category.
- List any 'blockers' or pending items for tomorrow.
OUTPUT: Create a clean Markdown file named 'Work_Report_2026-05-07.md'.
CONSTRAINTS: Use a formal business tone. Ensure the total time adds up correctly.
```

---

## ⚙️ План виконання

1. **Екстракція**: Claude виділить головні події та завдання з вашого "потоку свідомості" або сирих нотаток.
2. **Структурування**: Система впорядкує хаотичні записи у логічну послідовність.
3. **Оцінка**: Якщо ви забули вказати час, Claude запропонує реалістичну оцінку на основі складності завдання.

---

## ✅ Перевірка результату

- Перевірте, чи не пропустив Claude якісь важливі завдання.
- Переконайтеся, що тон звіту відповідає вашим відносинам з отримувачем (формальний чи дружній).
- Перевірте розділ "Плани на завтра" — він допоможе вам почати наступний день продуктивно.

---

## 💡 Поради

- **Голосові нотатки**: Ви можете використовувати транскрипцію ваших голосових нотаток як вхідні дані для цього воркфлоу.
- **Генерація інвойсів**: Цей журнал робіт може стати прямою основою для воркфлоу [Генерація інвойсів](invoice-generation.uk.md) в кінці місяця.
