# Створення контенту (Content Creation)

🌐 **Мови**: [English](content-creation.en.md) | [Français](content-creation.md) | [Українська 🇺🇦](content-creation.uk.md)

Цей воркфлоу перетворює Claude Cowork на вашу креативну команду, допомагаючи створювати якісний текстовий контент для бізнесу на основі ваших вхідних даних та досліджень.

---

## 📋 Контекст

Вам потрібно написати статтю в блог, опис продукту або новину для сайту. У вас є факти, але немає часу на те, щоб перетворити їх на цікавий текст.

## 🛠️ Підготовка

1. Підготуйте "сирі" факти, цифри або тези майбутнього тексту у файлі `notes.txt`.
2. Визначте цільову аудиторію та бажаний тон (formal, witty, inspirational).

---

## 📝 Промпт (Запит)

Скопіюйте цей запит у Claude:

```text
CONTEXT: I need to write a blog post for MacPlus Software about the benefits of AI automation.
TASK: Use the notes in 'ai_benefits_draft.txt' and research results in 'market_trends.md'. 
- Write a 1000-word article with an engaging title.
- Structure it with H2 and H3 headings.
- Include a list of 5 practical tips for small businesses.
- Add a Call to Action (CTA) at the end.
OUTPUT: Create a Markdown file 'Blog_Post_AI_2026.md'.
CONSTRAINTS: Use a 'Professional yet Accessible' tone. Ensure the content is SEO-optimized for the keyword 'AI automation for business'.
```

---

## ⚙️ План виконання

1. **Структурування**: Claude створить логічний план статті (вступ, основна частина, висновки).
2. **Написання**: Система розгорне ваші тези у зв'язний та цікавий текст.
3. **Оптимізація**: Claude перевірить текст на читабельність та вставить необхідні ключові слова для SEO.

---

## ✅ Перевірка результату

- Перевірте, чи зберіг Claude ваш стиль та основні ідеї.
- Оцініть заголовки — чи привертають вони увагу?
- Переконайтеся, що текст не звучить занадто "роботизовано" (за необхідності попросіть Claude додати більше живих прикладів).

---

## 💡 Поради

- **Перепакування (Repurposing)**: Попросіть Claude на основі написаної статті створити 3 коротких пости для соцмереж та сценарій для короткого відео (Reels/Shorts).
- **Візуалізація**: Попросіть Claude запропонувати описи для зображень (промпти для Midjourney або DALL-E), які ілюструватимуть вашу статтю.
