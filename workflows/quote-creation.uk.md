## Кейс використання

Вам потрібно створити професійну та точну комерційну пропозицію (кошторис) для клієнта. Цей воркфлоу допоможе вам:

- Сформувати перелік робіт та матеріалів з цінами.
- Розрахувати собівартість та бажану маржу.
- Згенерувати чіткі технічні описи, зрозумілі клієнту.
- Швидко адаптувати наявний шаблон під нового замовника.

> ⚠️ **Важливо**: Кошторис — це юридичне зобов'язання. Завжди перевіряйте ціни на матеріали (з урахуванням інфляції), обсяги та терміни перед відправкою.

## Покрокова інструкція

### Крок 1: Підготовка бази даних

```bash
# Створіть структуру
mkdir -p ~/Cowork-Workspace/{input,output/quotes}

# Якщо у вас є прайс-лист у Excel
cp ~/Documents/price-list.xlsx ~/Cowork-Workspace/input/
```

### Крок 2: Опис параметрів пропозиції

Передайте Cowork деталі для генерації:

```
Створи професійну комерційну пропозицію (кошторис):

КЛІЄНТ:
- Назва: [Назва компанії або ім'я]
- Об'єкт: [Адреса або назва проєкту]

МОЇ ДАНІ:
- Компанія: [Ваша назва/ФОП]
- ЄДРПОУ/ІПН: [Ваш номер]
- Контакти: [Телефон, Email]

ДЕТАЛІ:
- Номер пропозиції: КП-2026-001
- Дата: [Сьогодні]
- Термін дії цін: 14 днів

ПЕРЕЛІК ПОСЛУГ:
[Надайте список, наприклад:]
1. Монтаж кабелю (100 м)
2. Встановлення розеток (15 шт)
3. Щит електричний (1 шт)

Для кожного пункту додай:
- Чіткий опис робіт.
- Кількість та одиниці виміру.
- Ціну за одиницю та разом.

УМОВИ:
- Термін виконання: [X днів]
- Передоплата: [30/50/70%]
- Гарантія: [12 місяців]

Формат: PDF з логотипом.
Збережи у: ~/Cowork-Workspace/output/quotes/KP-2026-001.pdf
```

### Крок 3: Додавання технічних специфікацій (для будівництва)

```
Для кожного пункту кошторису додай технічні деталі:
- Використані матеріали (бренди, артикули, наприклад: "Кабель Одескабель ВВГнг-П").
- Відповідність стандартам (ДСТУ/ДБН).
- Етапи виконання робіт.

Приклад: "Монтаж теплої підлоги (Nexans), включаючи теплоізоляцію 30мм, монтажну стрічку та терморегулятор. Гарантія від виробника 20 років".
```

### Крок 4: Перевірка маржинальності та коригування

```
Допоможи розрахувати фінальну ціну:
- Собівартість матеріалів: [Сума]
- Витрати на логістику: [Сума]
- Трудовитрати: [К-сть годин] за ставкою [Сума/год]
- Бажана маржа: 25%

Розрахуй підсумкову суму для клієнта та сформуй фінальний PDF.
```

### Крок 5: Генерація та перевірка

```bash
open ~/Cowork-Workspace/output/quotes/KP-2026-001.pdf
```

## Варіанти пропозицій

### 1. Трьохрівневий кошторис (Economy / Standard / Premium)
Запропонуйте клієнту вибір:
- **Economy**: Бюджетні матеріали, базовий функціонал.
- **Standard**: Оптимальне співвідношення ціна/якість (рекомендовано).
- **Premium**: Топові бренди, розширена гарантія, пріоритетне обслуговування.

### 2. Кошторис з опціями
Виділіть базові роботи та додаткові опції, які клієнт може додати за бажанням (наприклад: "Розумне керування освітленням: +5500 грн").

## Найкращі практики для України

1. **Валютне застереження** — Якщо матеріали імпортні, додайте пункт: "Ціна зафіксована при курсі НБУ [курс] грн/USD. При зміні курсу більше ніж на 3%, сума за матеріали підлягає перерахунку".
2. **Унікальна нумерація** — Використовуйте систему `[Рік]-[Номер]`, щоб не плутатися в документах.
3. **Обмежений термін дії** — У зв'язку з коливанням цін на матеріали, ставте термін дії кошторису 3–7 днів.
4. **Чіткі межі робіт** — Прописуйте, що НЕ входить у вартість (наприклад: "Вивіз будівельного сміття не включено"), щоб уникнути конфліктів.
5. **Фото об'єкта** — Додавання фото "До" у комерційну пропозицію підвищує довіру клієнта.
