## Кейс використання

Вам потрібно створити детальні кошториси (комерційні пропозиції) для ваших клієнтів. Цей воркфлоу допоможе вам:

- Створювати професійні кошториси з цінами
- Розраховувати собівартість та маржу
- Генерувати чіткі технічні описи
- Швидко адаптувати існуючі кошториси для нових клієнтів

> ⚠️ **Важливо**: Кошторис передбачає вашу професійну відповідальність. Завжди перевіряйте ціни, кількість та терміни перед надсиланням.

## Покрокова інструкція

### Крок 1: Підготовка інформації

```bash
# Створення структури
mkdir -p ~/Cowork-Workspace/{input,output/quotes}

# Якщо у вас є прайс-лист
cp ~/Documents/price-list.xlsx ~/Cowork-Workspace/input/
```

### Крок 2: Створення кошторису

```
Створи професійний кошторис для:

КЛІЄНТ:
- Назва: [Назва клієнта]
- Проєкт: [Суть проєкту]
- Адреса робіт: [Адреса, якщо актуально]

МОЇ ДАНІ:
- Бізнес: [Ваша юридична назва]
- Адреса: [Ваша повна адреса]
- ІПН/ЄДРПОУ: [Ваш податковий номер]
- Контакт: [Телефон + email]

КОШТОРИС №: Q2026-001
Дата: 20 січня 2026 року
Термін дії: 3 місяці

ПОСЛУГИ:
[Детальний список із кількістю, описами та цінами]

Для кожного рядка:
- Точний опис
- Кількість
- Одиниця виміру
- Ціна за одиницю
- Разом

РОЗРАХУНКИ:
- Проміжний підсумок
- Податок (ПДВ/Податок з продажу)
- Разом

УМОВИ:
- Термін виконання: [X днів/тижнів]
- Депозит: [якщо актуально]
- Умови оплати: [умови]
- Термін дії: 3 місяці

Формат: PDF
Зберегти: ~/Cowork-Workspace/output/quotes/Q2026-001.pdf
```

### Крок 3: Додавання технічних описів

Для підрядників/будівництва:

```
Для кожної послуги додай:
- Детальний технічний опис
- Матеріали, що використовуються (бренди, артикули)
- Застосовні стандарти (ASTM, місцеві коди)
- Терміни проведення робіт
- Гарантії

Приклад:
"Встановлення 3-х металопластикових вікон (ПВХ), двокамерний склопакет, 4/16/4 аргон, 
коефіцієнт теплопередачі U=1.3 Вт/м².К, відповідність енергетичним стандартам, 
колір білий RAL 9016, згідно з місцевими будівельними нормами, 
гарантія 10 років, термін поставки 2 тижні"
```

### Крок 4: Перевірка та коригування

Cowork відобразить план. **Перевірте**:

- ✓ Правильність розрахунків (маржа врахована)
- ✓ Зрозумілість описів для клієнта
- ✓ Реалістичність термінів
- ✓ Чіткість умов

Якщо потрібні коригування:

```
Зміни кошторис:
- Збільш маржу на послугу 2 з 25% до 30%
- Скороти термін виконання з 4 до 3 тижнів
- Додай опцію: [опис опції] +[ціна]
```

### Крок 5: Генерація кошторису

```
Продовжуй генерацію кошторису
```

Перевірте результат:

```bash
open ~/Cowork-Workspace/output/quotes/Q2026-001.pdf
```

## Усунення несправностей

### Складність із розрахунком маржі

**Причина**: Погано оцінена база витрат
**Рішення**: Запитайте допомогу з розрахунками:

```
Мої витрати на матеріали: $450
Моя робота: 8 год @ $35/год
Витрати на проїзд: $40
Накладні витрати (20% від доходу): ?
Бажана маржа: 25%

Розрахуй мою ціну продажу
```

### Описи занадто технічні

**Причина**: Професійний жаргон
**Рішення**: Адаптуйте для клієнта:

```
Перепиши технічні описи мовою клієнта:
- Уникай жаргону
- Поясни вигоди (комфорт, економія, безпека)
- Залиш точність щодо норм у примітках дрібним шрифтом
```

### Кошторис занадто довгий

**Причина**: Занадто багато деталей
**Рішення**: Групуйте за етапами (лотами):

```
Згрупуй послуги за пакетами:
- Пакет 1: Чорнові роботи (підсумок)
- Пакет 2: Оздоблювальні роботи (підсумок)
- Пакет 3: Фінальні деталі (підсумок)

Деталі винеси в окремий додаток
```

## Найкращі практики

1. **Унікальна нумерація** — Система безперервної нумерації (Q[рік]-[номер])
2. **Обмежена дія** — Зазвичай 3 місяці (вказуйте чітко)
3. **Чіткі умови** — Депозит, терміни, оплата, гарантії
4. **Фото за потреби** — Додавайте фото об'єкта для кошторисів на реновацію
5. **Версії** — Зберігайте історію, якщо кошторис змінюється (v1, v2 тощо)
6. **Перевірка** — Попросіть когось переглянути документ у разі великої суми
7. **Відстеження** — Зв'яжіться з клієнтом через тиждень, якщо немає відповіді
8. **Кошторис = зобов'язання** — Ціни та послуги фіксуються, якщо не вказано інше
