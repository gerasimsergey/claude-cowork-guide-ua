# Воркфлоу: Пости в Соціальних Мережах

> **Орієнтовний час**: 10 хвилин на пост
> **Складність**: Початківець
> **Категорія**: 📣 Комунікація
> **Професії**: Торговці, Власники бізнесу, Фрілансери

🌐 **Мови**: [English](social-media-posts.en.md) | [Français](social-media-posts.md) | [Українська 🇺🇦](social-media-posts.uk.md)

## Кейс використання

Вам потрібно створювати цікавий та професійний контент для просування вашого бізнесу. Цей воркфлоу допоможе вам:

- Генерувати пости, адаптовані під різні платформи (Telegram, LinkedIn, Facebook, Instagram).
- Планувати контент-календар на місяць.
- Писати тексти, що спонукають до дії (CTA).
- Оптимізувати графік публікацій для максимального охоплення.

> ⚠️ **Ефект**: Регулярні публікації (3-5 на тиждень) підвищують впізнаваність бренду на 70% та залученість клієнтів на 50%.

## Попередні вимоги

- [ ] Cowork увімкнено в Claude Desktop.
- [ ] Акаунти в соцмережах (Telegram, LinkedIn, Facebook, Instagram).

- [ ] Створена робоча папка.

## Покрокова інструкція

### Крок 1: Визначення стратегії контенту

```bash
mkdir -p ~/Cowork-Workspace/social-media/{posts,images,calendar}
```

Встановіть рамки роботи:

```
Створи стратегію контенту для: [Назва бізнесу]

БІЗНЕС: [Наприклад: Електрик, Магазин квітів, ІТ-консультант]
ЦІЛЬОВА АУДИТОРІЯ: [B2B професіонали / Локальні споживачі / Змішана]

ПЛАТФОРМИ:

Telegram (Найпопулярніший в Україні):
- Аудиторія: Лояльні клієнти, швидкі новини.
- Тон: Неформальний, оперативний.
- Типи контенту: Опитування, швидкі поради, залаштунки.

LinkedIn:
- Аудиторія: B2B клієнти, партнери.
- Тон: Професійний, експертний.

Facebook/Instagram:
- Аудиторія: Масовий споживач, візуальний контент.
- Тон: Дружній, емоційний.

ПРАВИЛО ТРЕТИН (Content Pillars):
1/3 НАВЧАННЯ: Поради, "як зробити", тренди ринку.
2/3 ЗАЛУЧЕННЯ: Історії успіху, команда, "до та після".
3/3 ПРОДАЖІ: Спеціальні пропозиції, нові послуги, заклик до дії.

Збережи стратегію у: ~/Cowork-Workspace/social-media/strategy.txt
```

### Крок 2: Створення окремих постів

Згенеруйте контент для конкретної теми:

```
Створи пост для соцмереж на тему: [Тема, наприклад: Завершений ремонт кухні]

ВАРІАНТ ДЛЯ TELEGRAM (Коротко + Емодзі):
📸 Фото: [Опис]
Текст: "Ще один об'єкт здано! 🔨 Кухня для сім'ї в ЖК [Назва]. 
Що зробили: [список]. 
Вклалися в 2 тижні. Подобається результат? Став ❤️"

ВАРІАНТ ДЛЯ LINKEDIN (Професійно):
"Масштабуємо комфорт: завершено складний проєкт модернізації... [Деталі про технічні рішення]. Пишаємося командою!"

ВАРІАНТ ДЛЯ INSTAGRAM (Візуально):
"Гортайте, щоб побачити магію 'ДО та ПІСЛЯ' ✨. Яка деталь вам подобається найбільше? 👇"

Збережи варіанти у: ~/Cowork-Workspace/social-media/posts/kitchen-project.txt
```

### Крок 3: Планування контент-календаря

```
Створи контент-календар на [Місяць]:

ПОНЕДІЛОК: Мотивація / Плани на тиждень.
СЕРЕДА: Корисна порада (Навчання).
П'ЯТНИЦЯ: Кейс тижня (До та Після).
НЕДІЛЯ: Особисте / Команда.

Збережи у: ~/Cowork-Workspace/social-media/calendar/monthly-plan.xlsx
```

### Крок 4: Написання текстів, що чіпляють (Hooks)

```
Перепиши мій текст, використовуючи сильний "гачок" (Hook):

ПРИКЛАДИ ГАЧКІВ:
- Питання: "Ви знали, що 80% пожеж у квартирах стаються через стару проводку?"
- Цифра: "Як заощадити 2000 грн на комуналці за 1 годину."
- Провокація: "Припиніть купувати дешеві розетки!"

Використай один із них для посту про [Тема].
```

### Крок 5: Оптимізація часу та хештегів

```
Для України найкращий час для публікацій:
- Telegram: 09:00 - 10:00 та 18:00 - 20:00.
- LinkedIn: Вівторок-Четвер 10:00 - 12:00.

Згенеруй список з 10 актуальних хештегів для [Ваше місто] та [Ваша ніша].
Приклад: #ремонтКиїв #електрикЛьвів #бізнесУкраїна
```

## Найкращі практики

1. **Telegram — це база** — В Україні це найшвидший канал зв'язку. Використовуйте опитування та реакції для залучення.
2. **Якість фото** — Навіть на телефон знімайте при денному світлі. Чистий об'єкт на фото підвищує довіру на 100%.
3. **Відповідайте на коментарі** — Соцмережі люблять активність. Відповідь у перші 30 хвилин піднімає пост у стрічці.
4. **Менше тексту, більше користі** — Люди гортають стрічку швидко. Дайте головну думку в перших двох реченнях.
5. **Локальні хештеги** — Обов'язково додавайте назву вашого міста або району, щоб залучати місцевих клієнтів.
