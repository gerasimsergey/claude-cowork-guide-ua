# Воркфлоу: Відстеження Інвентарю та Залишків

> **Орієнтовний час**: 20 хвилин
> **Складність**: Середній
> **Категорія**: 🔧 Виробництво
> **Професії**: Торговці, Підрядники, Власники складів

🌐 **Мови**: [English](inventory-tracking.en.md) | [Français](inventory-tracking.md) | [Українська 🇺🇦](inventory-tracking.uk.md)

## Кейс використання

Вам потрібно контролювати запаси на складі чи в магазині, щоб уникати дефіциту товарів або, навпаки, затоварення. Цей воркфлоу допоможе вам:

- Створити систему обліку залишків.
- Встановити критичні рівні (Reorder Points) для автоматичних замовлень.
- Аналізувати оборотність товарів (ABC-аналіз).
- Оптимізувати капітал, заморожений у товарі.

> ⚠️ **Ефект**: Якісне управління запасами зменшує заморожений капітал на 20-30% та знижує ризик дефіциту на 40%.

## Покрокова інструкція

### Крок 1: Створення бази даних інвентарю

```bash
mkdir -p ~/Cowork-Workspace/inventory
```

Початкове наповнення системи:

```
Створи систему обліку інвентарю для: [Назва бізнесу]

СТРУКТУРА КАРТКИ ТОВАРУ:
- SKU: [Унікальний код]
- Назва: [Найменування]
- Категорія: [Група товарів]
- Постачальник: [Основний контакт]

ДАНІ ПРО ЗАЛИШКИ:
- Поточна кількість: [X одиниць]
- Одиниця виміру: [шт, м, кг, пакування]
- Місце зберігання: [Стелаж/Полиця]

ВАРТІСТЬ:
- Ціна закупівлі: [Сума] грн.
- Ціна продажу: [Сума] грн.

ПОРОГИ (Thresholds):
- Мінімальний залишок: [Точка замовлення]
- Максимальний залишок: [Ліміт складу]
- Оптимальна партія замовлення: [К-сть]

Приклад запису:
SKU: CAB-001
Назва: Кабель ВВГнг 3х1.5
Залишок: 450 м
Мінімум: 200 м (якщо менше — замовляти)

Збережи систему у: ~/Cowork-Workspace/inventory/master-inventory.xlsx
```

### Крок 2: Розрахунок точок замовлення (Reorder Points)

Налаштуйте Claude для аналізу попиту:

```
Розрахуй точки перебору (Reorder Points) для моїх товарів:

ФОРМУЛА:
Точка замовлення = (Середньодобове споживання × Термін доставки) + Страховий запас

ДАНІ ДЛЯ РОЗРАХУНКУ:
- Кабель: споживання 20м/день, доставка 3 дні, страховий запас 50м.
- Розрахунок: (20 * 3) + 50 = 110м.

РЕКОМЕНДАЦІЯ: Встанови "Мінімальний залишок" у системі на рівні 110м. Як тільки залишок впаде до цієї мети — Claude має видати попередження.

Збережи розрахунки у: ~/Cowork-Workspace/inventory/reorder-analysis.xlsx
```

### Крок 3: Реєстрація руху товарів (Прихід/Розхід)

```
Онови залишки за сьогодні [Дата]:

ПРИХІД (IN):
- SKU: CAB-001 | К-сть: +500м | Постачальник: Одескабель.

РОЗХІД (OUT):
- SKU: CAB-001 | К-сть: -120м | Об'єкт: Ремонт вул. Центральна.

ВТРАТИ/СПИСАННЯ:
- SKU: TILE-05 | К-сть: -2шт | Причина: Бій при розвантаженні.

Онови master-inventory.xlsx та згенеруй звіт про залишки.
```

### Крок 4: Аналіз оборотності та ABC-аналіз

Визначте, які товари приносять гроші, а які просто займають місце:

```
Проведи ABC-аналіз мого складу:

КАТЕГОРІЯ A (20% товарів = 80% обороту):
Найважливіші товари. Мають бути завжди в наявності. Суворий контроль.

КАТЕГОРІЯ B (30% товарів = 15% обороту):
Середня важливість.

КАТЕГОРІЯ C (50% товарів = 5% обороту):
Малоцінні товари, що повільно продаються. Мінімальний контроль.

Згенеруй звіт: Які товари варто вивести з асортименту, а на які збільшити страховий запас?
Збережи у: ~/Cowork-Workspace/inventory/abc-analysis.pdf
```

### Крок 5: Автоматизація сповіщень

```
Проаналізуй файл master-inventory.xlsx і виведи список товарів, які:
1. 🚨 КРИТИЧНО: Залишок < 50% від мінімуму.
2. ⚠️ ЗАМОВИТИ: Залишок <= Мінімальний залишок.
3. ✅ В НОРМІ: Залишок > Мінімальний.

Для товарів "ЗАМОВИТИ" згенеруй драфт імейла постачальнику.
```

## Найкращі практики

1. **Регулярна інвентаризація** — Раз на місяць робіть вибіркову перевірку (Spot Check) 5-10 випадкових позицій, щоб переконатися, що фізичний залишок збігається з цифровим.
2. **Правило FIFO (First In, First Out)** — Продавайте або використовуйте спочатку ті товари, які прийшли на склад раніше (особливо важливо для товарів з терміном придатності).
3. **Маркування SKU** — Навіть якщо у вас немає сканера штрих-кодів, підпишіть кожну полицю відповідним кодом SKU. Це зменшить помилки при пошуку.
4. **ABC-пріоритети** — Не витрачайте однакову кількість часу на облік цвяхів (категорія C) та дорогих інструментів (категорія A).
5. **Враховуйте сезонність** — Збільшуйте страхові запаси перед початком сезону (наприклад, будівельні суміші навесні).
