# Воркфлоу: Порівняння цін постачальників

> **Орієнтовний час**: 20 хвилин
> **Складність**: Початкова
> **Категорія**: 📋 Адміністративні
> **Професії**: Підрядники, Торговці, Малий бізнес

🌐 **Мови**: [English](price-comparison.en.md) | [Français](price-comparison.md) | [Українська 🇺🇦](price-comparison.uk.md)

---

## Кейс використання

Вам потрібно порівняти ціни від кількох постачальників, щоб прийняти найкраще рішення про закупівлю. Цей воркфлоу допоможе вам:

- Порівнювати пропозиції від кількох постачальників
- Оцінювати загальну вартість (не лише ціну за одиницю)
- Застосовувати зважену оцінку за кількома критеріями
- Приймати рішення на основі даних
- Вести історію ефективності постачальників

> ⚠️ **Важливо**: Ціна — це лише один із факторів. При прийнятті рішення враховуйте якість, надійність, терміни доставки та умови оплати.

### Крок 4: Врахування повної вартості володіння

```
АНАЛІЗ ПОВНОЇ ВАРТОСТІ (крім ціни в пропозиції):

СЦЕНАРІЙ: Закупівля офісного приладдя у 3-х постачальників

ПОСТАЧАЛЬНИК А — Найдешевша пропозиція ($1,180):
+ Вартість товару: $1,180
+ Доставка: $25 (повільна, 7-10 днів)
+ Комісія за терміновість, якщо знадобиться: $75 (поширена проблема)
+ Повернення: $15 комісія за кожне повернення
+ Оплата: Готівка при отриманні (немає вигоди для грошового потоку)
= Орієнтовна реальна вартість: $1,180-1,295 (залежно від проблем)

ПОСТАЧАЛЬНИК Б — Середня пропозиція ($1,220):
+ Вартість товару: $1,220
+ Доставка: БЕЗКОШТОВНО (при замовленні понад $1,000)
+ Термінова доставка: БЕЗКОШТОВНЕ оновлення за потреби
+ Повернення: БЕЗКОШТОВНО, передплачена етикетка
+ Оплата: Відстрочка 30 днів (вигода для грошового потоку)
+ Програма лояльності: 2% повернення = -$24
= Орієнтовна реальна вартість: $1,196

ПОСТАЧАЛЬНИК В — Вища пропозиція ($1,250):
+ Вартість товару: $1,250
+ Доставка: Включена у вартість
+ Терміновість: Завжди відвантажує наступного дня (стандарт)
+ Повернення: Без проблем, без комісій
+ Оплата: Відстрочка 45 днів (краще для грошового потоку)
+ Оптова знижка: -5% = -$62.50
= Орієнтовна реальна вартість: $1,187.50

ВИСНОВОК: Постачальник В здається найдорожчим, але насправді є конкурентним, 
якщо врахувати загальні витрати + нематеріальні чинники (заощаджений час, надійність)

РЕКОМЕНДАЦІЯ: Постачальник В для регулярних закупівель, Постачальник А для одноразових

Додай цей аналіз до таблиці порівняння (АРКУШ 3: АНАЛІЗ ПОВНОЇ ВАРТОСТІ)
```

## Усунення несправностей

### Пропозиції в різних форматах/одиницях

**Причина**: Постачальники вказують ціни за різні одиниці або упаковки
**Рішення**: Приведіть до спільної одиниці виміру:

```
ПРИКЛАД НОРМАЛІЗАЦІЇ:

ТОВАР: Фарба

Постачальник А: $45 за галон
Постачальник Б: $180 за 5-галонне відро
Постачальник В: $540 за упаковку (12 галонів)

НОРМАЛІЗАЦІЯ ДО: $ за галон
- Постачальник А: $45/гал
- Постачальник Б: $180 ÷ 5 = $36/гал ✅ НАЙКРАЩЕ
- Постачальник В: $540 ÷ 12 = $45/гал

ТАКОЖ ВРАХУЙТЕ:
- Чи можете ви купити в упаковці Постачальника Б? (Чи потрібно вам 5+ галонів?)
- Зберігання: Чи є де зберігати оптові закупівлі?
- Термін придатності: Чи не зіпсується товар до використання?

Якщо потрібно рівно 3 галони:
- Постачальник А: 3 × $45 = $135 ✅ НАЙКРАЩЕ (без залишків)
- Постачальник Б: 1 відро (5 гал) = $180 (2 галони залишку)

РЕКОМЕНДАЦІЯ: Залежить від необхідної кількості та можливостей зберігання
```

### Неспівставні товари

**Причина**: Запропоновані товари не є ідентичними (різна якість/характеристики)
**Рішення**: Коригуйте ціну з урахуванням різниці в якості:

```
ПРИКЛАД: Професійне обладнання проти побутового

Постачальник А: Побутова модель — $500
- Гарантія: 1 рік
- Очікуваний термін служби: 3-5 років
- Обслуговування: Силами користувача

Постачальник Б: Професійна модель — $800
- Гарантія: 3 роки
- Очікуваний термін служби: 8-10 років
- Обслуговування: Включено сервісний план

ПОРІВНЯННЯ "ЯБЛУКА ДО ЯБЛУК":

Загальна вартість побутової моделі за 10 років:
- Купівля: $500 × 2 (заміна через 5 років) = $1,000
- Обслуговування: ~$200 (в середньому)
- Витрати через простої: ~$150 (оціночно)
= $1,350 за 10 років = $135/рік

Загальна вартість професійної моделі за 10 років:
- Купівля: $800 × 1 = $800
- Обслуговування: $0 (включено)
- Простої: Мінімальні завдяки надійності
= $800 за 10 років = $80/рік

КОРИГОВАНЕ ПОРІВНЯННЯ: Професійна модель на 41% дешевша за термін служби
РЕКОМЕНДАЦІЯ: Професійна модель (Постачальник Б), попри вищу початкову ціну
```

## Найкращі практики

1. **Отримуйте 3+ пропозиції** — Мінімум 3 постачальники для змістовного порівняння
2. **Нормалізуйте одиниці виміру** — Приводьте всі ціни до однієї одиниці для чесного порівняння
3. **Включайте всі витрати** — Доставку, податки, збори, а не лише ціну за одиницю
4. **Зважуйте критерії** — Ціна — це не все; враховуйте доставку, якість, сервіс
5. **Документуйте рішення** — Фіксуйте, чому ви обрали саме цього постачальника (корисно для майбутнього)
6. **Відстежуйте ефективність** — Оцінюйте постачальників після доставки
7. **Будуйте стосунки** — Пріоритетні постачальники з часом часто пропонують кращі умови
8. **Ведіть переговори** — Використовуйте конкурентні пропозиції як важіль ("Постачальник Б запропонував $X...")
