## Кейс використання

Вам потрібно закупити матеріали або обладнання, і у вас є декілька пропозицій від різних постачальників. Цей воркфлоу допоможе вам:

- Порівняти ціни на окремі позиції та загальну суму закупівлі.
- Врахувати приховані витрати (доставка, розвантаження, податки).
- Оцінити постачальників за декількома критеріями (ціна, терміни, умови оплати).
- Прийняти обґрунтоване рішення на основі даних, а не емоцій.

> **Важливо**: Ціна — це лише один із факторів. Надійність постачальника та швидкість доставки часто важливіші за економію в декілька відсотків.

- [ ] Створена робоча папка.

## Покрокова інструкція

### Крок 1: Збір пропозицій

```bash
mkdir -p ~/Cowork-Workspace/{input/quotes,output/comparisons}

# Скопіюйте файли пропозицій
cp ~/Documents/Quotes/Supplier*.pdf ~/Cowork-Workspace/input/quotes/
```

### Крок 2: Створення порівняльної таблиці

Попросіть Cowork проаналізувати файли:

```
Порівняй пропозиції на закупівлю [Назва товарів]:

ВХІДНІ ФАЙЛИ:
- Постачальник А: ~/Cowork-Workspace/input/quotes/supplier-a.pdf
- Постачальник Б: ~/Cowork-Workspace/input/quotes/supplier-b.pdf

З кожного файлу витягни:
1. Назву товару та кількість.
2. Ціну за одиницю (з ПДВ та без).
3. Вартість доставки.
4. Термін поставки (днів).
5. Умови оплати (передоплата/відстрочка).

СТВОРИ ТАБЛИЦЮ EXCEL:
~/Cowork-Workspace/output/comparisons/price-comparison-[date].xlsx
З умовним форматуванням: виділи найнижчу ціну за кожну позицію зеленим кольором.
```

### Крок 3: Розрахунок повної вартості (Total Cost)

Іноді дешевий товар дорожчий у доставці:

```
На основі таблиці розрахуй повну вартість закупівлі для кожного постачальника:
- Сума товарів + Доставка + Комісія за переказ коштів.
- Розрахуй "Ціну за одиницю з урахуванням логістики".

Додай висновок: Який постачальник найвигідніший за фінальною сумою?
```

### Крок 4: Матриця зваженої оцінки (Scoring)

Якщо ціни схожі, оцініть інші фактори (від 1 до 10):

```
Проведи зважену оцінку постачальників:

КРИТЕРІЇ (Вага):
1. Ціна (50%)
2. Швидкість доставки (30%)
3. Надійність/Гарантія (20%)

Оціни Постачальника А та Постачальника Б за цими критеріями.
Хто набрав більше балів?
```

### Крок 5: Звіт про вибір постачальника

```
Згенеруй фінальний звіт (Decision Report):
- Хто обраний?
- Обґрунтування (наприклад: "Обрано Постачальника Б, бо хоча ціна на 2% вища, термін доставки вдвічі швидший, що критично для проєкту").
- Розрахована економія порівняно з найдорожчим варіантом.

Збережи як: ~/Cowork-Workspace/output/comparisons/decision-report.pdf
```

## Найкращі практики

1. **Порівнюйте ідентичне** — Переконайтеся, що ви порівнюєте товари однакової якості та характеристик (наприклад, однакова марка бетону або клас енергоефективності).
2. **Враховуйте ПДВ** — Один постачальник може вказати ціну з ПДВ, інший — без. Завжди зводьте все до єдиного показника.
3. **Перевіряйте наявність** — Завжди уточнюйте, чи товар дійсно є на складі, перш ніж приймати рішення на основі ціни.
4. **Вартість грошей** — Відстрочка платежу на 30 днів часто вигідніша, ніж знижка 2% при повній передоплаті. Попросіть Cowork розрахувати вигоду відтермінування.
5. **Історія відносин** — Якщо ви постійний клієнт, використовуйте пропозицію конкурента як аргумент для отримання додаткової знижки у вашого постійного постачальника.
