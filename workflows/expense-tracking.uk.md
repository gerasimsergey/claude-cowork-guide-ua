## Кейс використання

У вас є пачка фотофіскальних чеків, скріншотів переказів у Privat24/Mono або PDF-рахунків, і вам потрібно:

- Витягнути дані про суми, дати та постачальників у таблицю Excel.
- Автоматично розподілити витрати за категоріями (матеріали, пальне, обіди тощо).
- Підготувати звіт для бухгалтера або замовника.
- Відстежити залишок бюджету по проєкту.

> ⚠️ **Важливо**: Claude чудово зчитує основні поля (сума, дата, назва магазину), але може помилятися у складних таблицях всередині чека. Завжди перевіряйте фінальну суму за звітом.

## Покрокова інструкція

### Крок 1: Збір та підготовка чеків

```bash
mkdir -p ~/Cowork-Workspace/input/receipts
mkdir -p ~/Cowork-Workspace/output
```

Завантажте всі ваші фото або PDF у папку `receipts`. Намагайтеся, щоб текст на фото був чітким та освітленим.

### Крок 2: Оцифрування даних

Попросіть Claude витягнути дані:

```
Проаналізуй усі файли в ~/Cowork-Workspace/input/receipts/ та створи Excel-таблицю.

КОЛОНКИ:
- Дата (РРРР-ММ-ДД)
- Постачальник (Назва магазину/ФОП)
- Сума (Число)
- Валюта (UAH)
- Категорія (Матеріали, Логістика, Харчування, Інструмент)
- ПДВ (якщо вказано в чеку окремо)
- Назва файлу (для перевірки)

Збережи у: ~/Cowork-Workspace/output/vitraty.xlsx
```

### Крок 3: Створення звітів та підсумків

```
На основі файлу vitraty.xlsx створи фінансове зведення:
1. Скільки всього витрачено грн?
2. Розподіл витрат у % за категоріями (Зроби таблицю).
3. Топ-3 найдорожчі закупівлі.
4. Які чеки були нечіткими та потребують ручної перевірки?

Збережи підсумок у: ~/Cowork-Workspace/output/summary.txt
```

### Крок 4: Формування звіту про відшкодування (Reimbursement)

Якщо ви купували щось за власні кошти для компанії:

```
Сформуй PDF-звіт про відшкодування витрат за [Місяць]:
- ПІБ: [Ваше ім'я]
- Проєкт: [Назва]
- Таблиця з датами та сумами.
- Місце для підпису керівника.
```

### Крок 5: Перевірка на дублікати

```
Перевір, чи немає серед оцифрованих чеків дублікатів (однакова дата, сума та магазин). Виділи їх у звіті.
```

## Регіональні особливості (Україна)

1. **Валюта** — Використовуйте символ ₴ або код UAH.
2. **ПДВ** — Для компаній на загальній системі важливо виділяти суму ПДВ (20%) із чеків.
3. **Фіскальний номер** — Ви можете просити Claude записувати фіскальний номер чека (ФН) для подальшої перевірки на сайті ДПС.
4. **РРО/ПРРО** — Клод розпізнає чеки сучасних програмних РРО (Checkbox, Вчасно тощо).

---

## Найкращі практики

1. **Якість фото** — Знімайте чек зверху, без тіней та під прямим кутом. Чим чіткіший текст, тим вища точність оцифрування (до 98%).
2. **Обробка пачками** — Не намагайтеся оцифрувати 100 чеків за один раз. Краще робити це групами по 10-15 файлів.
3. **Категоризація** — Якщо Claude помилився з категорією (наприклад, "Епіцентр" записав у "Харчування"), дайте уточнення: "Усі чеки з Епіцентру — це Будматеріали".
4. **Зберігайте паперові оригінали** — Навіть якщо у вас є цифрова копія, для податкової перевірки в Україні часто потрібні фізичні чеки (якщо вони фіскальні).
5. **Автоматизація** — Налаштуйте щотижневе оновлення таблиці витрат, щоб не робити все в останню ніч перед звітом.
