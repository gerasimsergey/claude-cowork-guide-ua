## Кейс використання

У вас є фотографії чеків, скріншоти або скановані документи, і вам потрібно:

- Вилучити дані про витрати в електронну таблицю
- Категоризувати витрати
- Створити підсумки та резюме
- Відстежити витрати для відшкодування або бухгалтерського обліку

> ⚠️ **Важливо**: Claude чудово зчитує **поля** чеків (продавець, дата, сума), але слабше справляється з **попозиційним вилученням** із таблиць (63% точності проти 97% для основних полів). Завжди перевіряйте підсумкові суми вручну. Виділяйте 30-50% часу, заощадженого на рутині, для перевірки та виправлення помилок.

## Попередні вимоги

## Покрокова інструкція

### Крок 1: Підготовка чеків

```bash
# Створіть структуру робочого простору
mkdir -p ~/Cowork-Workspace/{input/receipts,output}

# Скопіюйте чеки в робочий простір
cp ~/Downloads/*.jpg ~/Cowork-Workspace/input/receipts/
cp ~/Downloads/*.png ~/Cowork-Workspace/input/receipts/
cp ~/Downloads/*.pdf ~/Cowork-Workspace/input/receipts/
```

**Порада**: Називайте файли описово, якщо це можливо (напр., `2024-01-15-uber.jpg`).

### Крок 2: Перевірка якості чеків

Перевірте якість зображень — Cowork використовує OCR (оптичне розпізнавання символів) і найкраще працює з:
- Чіткими фото у фокусі
- Гарним освітленням
- Повністю видимим чеком
- Мінімумом відблисків/тіней

### Крок 3: Початкове вилучення даних

Почніть із базового вилучення:

```
Вилучи дані про витрати із зображень чеків у ~/Cowork-Workspace/input/receipts/

Створи Excel-файл із такими колонками:
- Date (формат РРРР-ММ-ДД)
- Vendor/Store (Продавець)
- Amount (тільки число, без символу валюти)
- Currency (3-літерний код: USD, EUR, UAH тощо)
- Category (Категорія)
- Payment Method (якщо видно: Готівка, Картка)
- Receipt Filename (назва файлу для довідки)

Зберегти у: ~/Cowork-Workspace/output/expenses.xlsx
```

### Крок 4: Перевірка результатів

Відкрийте Excel-файл і перевірте:
- Чи правильні дати?
- Чи точні суми?
- Чи правильно розпізнані продавці?
- Чи немає пропущених або спотворених даних?

### Крок 5: Запит на виправлення

Якщо є помилки:

```
Переглянь створений файл витрат.

Потрібні виправлення:
- Рядок 3: Сума має бути 45.99, а не 4599
- Рядок 7: Дата має бути 2024-01-15
- Рядок 12: Продавець "Starbucks", а не "Slarbucks"

Зроби ці виправлення та збережи файл.
```

### Крок 6: Додавання підсумків

Доповніть таблицю аналізом:

```
Доповни ~/Cowork-Workspace/output/expenses.xlsx такими листами:

Лист 2 — "Підсумок за категоріями":
- Категорія
- Загальна сума
- Кількість транзакцій
- Відсоток від загального обсягу

Лист 3 — "Місячний підсумок":
- Місяць
- Загальна сума
- Кількість транзакцій
- Топ-категорія

Лист 4 — "Підсумок за продавцями":
- Продавець
- Усього витрачено
- Кількість візитів
- Середній чек

Додай формули, щоб підсумки оновлювалися автоматично.
Використовуй [європейський/американський] синтаксис формул.
```

### Крок 7: Фінальний огляд

```
Створи фінальний звіт за даними витрат:
- Загальна сума витрат
- Охоплений діапазон дат
- Кількість оброблених чеків
- Чеки, з яких не вдалося вилучити дані повністю

Зберегти звіт у ~/Cowork-Workspace/output/expense-summary.txt
```

## Приклади промптів

### Базове вилучення
```
Вилучи витрати з чеків у ~/Cowork-Workspace/input/receipts/
Колонки: Дата, Продавець, Сума, Категорія
Зберегти у ~/Cowork-Workspace/output/expenses.xlsx
```

### З автоматичною категоризацією
```
Вилучи витрати та автоматично категоризуй їх:
- Харчування: ресторани, кафе, продукти
- Транспорт: таксі, пальне, паркування
- Офіс: канцтовари, ПЗ, підписки
- Подорожі: готелі, квитки, оренда авто
- Інше: усе інше

Зберегти з категоріями у ~/Cowork-Workspace/output/categorized-expenses.xlsx
```

### Звіт для відшкодування
```
Створи звіт на відшкодування на основі витрат у ~/Cowork-Workspace/output/expenses.xlsx

Сформуй документ Word із:
- Ім'я співробітника: [Ваше ім'я]
- Період звіту: [Визначити за датами]
- Таблиця деталізованих витрат
- Підсумки за категоріями
- Загальна сума до виплати
- Місце для підпису та дата

Зберегти у ~/Cowork-Workspace/output/reimbursement-report.docx
```

## Регіональні налаштування

### Формат для України/Європи
```
Використовуй європейський формат:
- Дати: ДД.ММ.РРРР
- Валюта: UAH/EUR
- Розділювач у формулах: крапка з комою (;)
```

### Змішані валюти
```
Оброби кілька валют:
- Залиш колонку з оригінальною валютою
- Додай колонку "Сума в USD" з конвертацією
- Використовуй курс обміну на дату чека
- Зазнач курс конвертації в підсумку
```

## Найкращі практики

1. **Добре називайте файли чеків** — `РРРР-ММ-ДД-продавець.jpg` допомагає Cowork.
2. **Обробляйте партіями** — не намагайтеся обробити 200 чеків за раз; робіть по 20-30.
3. **Перевіряйте ВСІ підсумки** — точність OCR для окремих позицій становить ~63%; завжди перевіряйте суми вручну.
4. **Зберігайте оригінали** — не видаляйте зображення чеків до повної перевірки.
5. **Позначайте сумніви** — просіть Cowork помічати рядки, де він не впевнений у розпізнаванні.
6. **Виділяйте час на валідацію** — плануйте 30-50% "заощадженого" часу на виправлення помилок.
