## Кейс використання

У вас є захаращена папка зі змішаними файлами — папка "Завантаження", папка проєкту або будь-яка неорганізована колекція. Цей воркфлоу допоможе вам:

- Сортувати файли за типом, датою або проєктом
- Створювати логічні структури папок
- Генерувати звіти про організацію
- Виправляти непослідовність у назвах файлів

> ⚠️ **Встановлюйте реалістичні очікування**: Незалежних тестів продуктивності ШІ в організації файлів не існує. Заяви розробників про "5-кратну економію часу" — це маркетинг, а не дослідження. ШІ найбільше допомагає з **рутинною категоризацією**; складні або неоднозначні файли все одно потребують оцінки людиною.

## Покрокова інструкція

### Крок 1: Підготовка робочого простору

```bash
# Створіть робочий простір, якщо потрібно
mkdir -p ~/Cowork-Workspace/{input,output}

# Скопіюйте файли для організації (поки що не переміщуйте оригінали)
cp -R ~/Downloads/* ~/Cowork-Workspace/input/

# АБО для конкретної папки
cp -R ~/МояЗахаращенаПапка/* ~/Cowork-Workspace/input/
```

**Важливо**: Спочатку копіюйте, а не переміщуйте. Це збереже оригінали, поки ви не будете задоволені результатом.

### Крок 2: Огляд файлів

Почніть діалог із Cowork і попросіть надати огляд:

```
Склади список усіх файлів у ~/Cowork-Workspace/input/ і покажи мені:
- Загальну кількість
- Присутні типи файлів
- Орієнтовний розподіл за типами
- Будь-які закономірності в назвах
```

Перегляньте це, перш ніж рухатися далі. Це допоможе зрозуміти, з чим ви працюєте.

### Крок 3: Вибір стратегії організації

**Варіант А: Організація за типом**
```
Упорядкуй усі файли в ~/Cowork-Workspace/input/ за типом файлу.

Створи такі папки:
- Documents (pdf, doc, docx, txt, md, rtf)
- Spreadsheets (xls, xlsx, csv, numbers)
- Images (jpg, jpeg, png, gif, svg, webp)
- Videos (mp4, mov, avi, mkv)
- Audio (mp3, wav, m4a, flac)
- Archives (zip, rar, tar, gz, 7z)
- Code (js, py, html, css, json, xml)
- Other (усе інше)

Перемісти файли у відповідні папки.
Збережи звіт про організацію у ~/Cowork-Workspace/output/organization-report.txt
```

**Варіант Б: Організація за датою**
```
Упорядкуй усі файли в ~/Cowork-Workspace/input/ за датою.

Створи структуру папок рік/місяць:
- 2024/
  - 01-Січень/
  - 02-Лютий/
  - ...
- 2025/
  - 01-Січень/
  - ...

Використовуй дату зміни файлу для сортування.
Перемісти файли в папки.
Створи звіт у ~/Cowork-Workspace/output/date-organization.txt
```

**Варіант В: Організація за проєктом**
```
Проаналізуй файли в ~/Cowork-Workspace/input/ та упорядкуй їх за проєктами.

Ідентифікуй проєкти на основі:
- Шаблонів у назвах файлів
- Змісту (прочитай текстові файли)
- Пов'язаних груп файлів

Створи папку для кожного ідентифікованого проєкту.
Поклади файли, які неможливо класифікувати, у папку "Unsorted/".
Створи індекс проєктів у ~/Cowork-Workspace/output/project-index.md
```

### Крок 4: Перегляд плану

Cowork покаже заплановані дії. **Читайте уважно**:

- Чи відповідає структура папок вашим очікуванням?
- Чи правильно визначено категорії файлів?
- Чи підходить обсяг робіт?

Якщо щось виглядає неправильно, скажіть:
```
Зачекай. Зміни план:
- [Ваше виправлення]
```

### Крок 5: Виконання та перевірка

Якщо план виглядає добре:
```
Виконуй план
```

Після завершення:
```bash
# Перевірте результати
ls -la ~/Cowork-Workspace/input/

# Перегляньте звіт
cat ~/Cowork-Workspace/output/organization-report.txt
```

### Крок 6: Застосування до оригіналів (опційно)

Якщо ви задоволені результатом:
```bash
# Видаліть оригінальну захаращену папку (ОБЕРЕЖНО!)
rm -rf ~/Downloads/*

# Перемістіть упорядковані файли на початкове місце
mv ~/Cowork-Workspace/input/* ~/Downloads/
```

## Усунення несправностей

### "Permission denied" (відмовлено в доступі) для деяких файлів

```bash
# Перевірте права доступу до файлів
ls -la ~/Cowork-Workspace/input/

# Виправте права, якщо потрібно
chmod 644 ~/Cowork-Workspace/input/*
```

### Файли класифіковані неправильно

Будьте точнішими в правилах:
```
Перекласифікуй: перемісти файли .md у Документи, а не в Код.
Файли .json зі словом "config" у назві перемісти в папку Configuration.
```

### Приховані файли ігноруються

```
Також оброби приховані файли (що починаються з .)
Включи їх у відповідні категорії.
```

### Занадто багато файлів у папці "Other" (Інше)

Визначте додаткові категорії:
```
Створи додаткові папки:
- Fonts (ttf, otf, woff)
- Installers (dmg, pkg, exe)
- Data (json, xml, yaml)
```

## Варіації

### Очищення назв файлів під час організації

```
Упорядкуй за типом ТА стандартизуй назви файлів:
- Видали спеціальні символи
- Заміни пробіли на дефіси
- Переведи в нижній регістр
- Додай префікс із датою (РРРР-ММ-ДД)
```

### Пошук дублікатів

```
Перед організацією знайди дублікати файлів.
Створи звіт про дублікати в ~/Cowork-Workspace/output/duplicates.txt
Згрупуй дублікати, але нічого не видаляй.
Потім упорядкуй унікальні файли за типом.
```

### Архівування старих файлів

```
Упорядкуй нещодавні файли (за останні 6 місяців) за типом.
Перемісти старіші файли в ~/Cowork-Workspace/input/archive/
Створи маніфест заархівованих файлів.
```

## Найкращі практики

1. **Завжди спочатку копіюйте** — працюйте з копіями, поки не будете задоволені результатом.
2. **Перевіряйте план** — не схвалюйте дії автоматично.
3. **Вивчайте звіт** — перевіряйте точність категоризації.
4. **Починайте з малих тестів** — протестуйте на 20 файлах, перш ніж обробляти 2000.
5. **Зберігайте бекапи** — поки не будете впевнені в результатах.
6. **Виділяйте час на перевірку** — плануйте 30-50% часу на перевірку та виправлення неправильно розподілених файлів.
7. **Стежте за квотою** — обробка великих пакетів (500+ файлів) споживає багато токенів.
