## Кейс використання

У вас є безлад у папці "Завантаження", на робочому столі або в папці проєкту, де змішані різні типи файлів. Цей воркфлоу допоможе вам:

- Відсортувати файли за типами (PDF, фото, кошториси, договори).
- Створити логічну структуру папок за роками, місяцями або клієнтами.
- Виправити хаотичні назви файлів (наприклад, "IMG_2342.jpg" → "2026-01-20_Об'єкт_Київ.jpg").
- Створити звіт про вміст папок.

> ⚠️ **Важливо**: ШІ найкраще справляється з рутинною класифікацією за розширеннями або шаблонами імен. Якщо файли мають складний зміст, перевіряйте результат сортування.

## Покрокова інструкція

### Крок 1: Підготовка робочого простору

```bash
# Створіть робочу папку
mkdir -p ~/Cowork-Workspace/input
mkdir -p ~/Cowork-Workspace/output

# Скопіюйте файли для тесту (не переміщуйте оригінали відразу!)
cp -R ~/Downloads/щось_незрозуміле/* ~/Cowork-Workspace/input/
```

### Крок 2: Огляд файлів

Попросіть Claude проаналізувати хаос:

```
Проаналізуй вміст папки ~/Cowork-Workspace/input/ та надай звіт:
- Скільки всього файлів?
- Які типи файлів присутні (PDF, JPG, XLSX тощо)?
- Чи є закономірності в назвах?
```

### Крок 3: Вибір стратегії сортування

**Варіант А: Сортування за ТИПОМ (найпростіше)**
```
Розподіли файли в ~/Cowork-Workspace/input/ за папками:
- Документи (pdf, docx, txt)
- Кошториси та Таблиці (xlsx, csv)
- Фото та Відео (jpg, mp4)
- Архіви (zip, rar)

Збережи звіт про сортування у: ~/Cowork-Workspace/output/report.txt
```

**Варіант Б: Сортування за ПРОЄКТОМ/КЛІЄНТОМ**
```
Проаналізуй назви файлів у ~/Cowork-Workspace/input/. 
Створи окремі папки для кожного проєкту, який ти ідентифікуєш (наприклад: "ЖК_Центральний", "ФОП_Іванов").
Файли без чіткої прив'язки залиш у папці "Інше".
```

**Варіант В: Сортування за ДАТОЮ**
```
Створи структуру папок РІК / МІСЯЦЬ та розклади файли за датою їх створення.
Наприклад: 2026 / 01-Січень / ...
```

### Крок 4: Перейменування файлів (Standardization)

```
Перейменуй файли в папці [Назва] за стандартом: 
РРРР-ММ-ДД_Опис.розширення
Наприклад: 2026-05-15_Акт_виконаних_робіт.pdf

Видали з назв зайві символи, пробіли заміни на підкреслення (_).
```

### Крок 5: Перевірка та завершення

```bash
# Перевірте результат у папці input
ls -R ~/Cowork-Workspace/input/

# Якщо все влаштовує, перемістіть у постійне місце
mv ~/Cowork-Workspace/input/* ~/Documents/Мій_Порядок/
```

## Варіації

### Пошук дублікатів
```
Перш ніж сортувати, знайди файли з однаковими назвами або розміром. 
Склади список дублікатів, але нічого не видаляй без мого підтвердження.
```

### Архівування старих файлів
```
Знайди всі файли, які не відкривалися більше 1 року. 
Перемісти їх у папку "Архів_2025" та створи текстовий список вмісту.
```

## Найкращі практики

1. **Завжди копіюйте, а не переміщуйте** — Поки ви не впевнені в логіці сортування ШІ, працюйте з копіями.
2. **Датуйте все** — Формат РРРР-ММ-ДД дозволяє файлам автоматично сортуватися за хронологією в будь-якій системі.
3. **Уникайте глибоких вкладень** — Намагайтеся не робити більше 3-4 рівнів папок (напр: Проєкт > Рік > Тип_файлу). Глибші структури важко шукати.
4. **Використовуйте зрозумілі назви** — "Рахунок_45.pdf" гірше ніж "Рахунок_45_Бетон_Одескабель.pdf".
5. **Регулярність** — Виділяйте 10 хвилин наприкінці кожного тижня для "цифрового прибирання" за допомогою цього воркфлоу.
