# Воркфлоу: Перетворення Кошторису в Інвойс

> **Орієнтовний час**: 10 хвилин
> **Складність**: Новачок

🌐 **Мови**: [English](quote-to-invoice.en.md) | [Français](quote-to-invoice.md) | [Українська 🇺🇦](quote-to-invoice.uk.md)

## Кейс використання

Клієнт прийняв ваш кошторис (комерційну пропозицію), і вам потрібно швидко перетворити його на інвойс (рахунок). Цей воркфлоу допоможе вам:

- Автоматично конвертувати кошторис в інвойс.
- Зберегти послідовність нумерації та сум.
- Додати юридичні реквізити, необхідні для інвойсу.
- Коректно архівувати документи.

> ⚠️ **Важливо**: Завжди перевіряйте, чи відповідає прийнятий кошторис фактично наданим послугам в інвойсі.

---

## Попередні вимоги

- [ ] Cowork увімкнено в Claude Desktop.
- [ ] Прийнятий кошторис (PDF або Word).
- [ ] Наступний номер інвойсу.
- [ ] Підтвердження клієнта (імейл, підпис або усне підтвердження).
- [ ] Створена робоча папка.

## Покрокова інструкція

### Крок 1: Підготуйте вихідний кошторис

```bash
mkdir -p ~/Cowork-Workspace/{input,output/invoices}

# Скопіюйте прийнятий кошторис у папку input
cp ~/Documents/Quotes/Q2026-042.pdf ~/Cowork-Workspace/input/
```

### Крок 2: Конвертація в інвойс

```
Перетвори кошторис ~/Cowork-Workspace/input/Q2026-042.pdf на інвойс

ТРАНСФОРМАЦІЇ:
1. Заміни "КОШТОРИС" на "ІНВОЙС" (або "РАХУНОК-ФАКТУРА")
2. Заміни номер кошторису Q2026-042 на INV-2026-085
3. Заміни дату кошторису на дату інвойсу: 20 січня 2026 року
4. Додай термін оплати: 19 лютого 2026 року (30 днів)

ЗАЛИШ БЕЗ ЗМІН:
- Реквізити клієнта
- Послуги та кількість
- Ціни за одиницю та підсумки (сума, ПДВ, разом)
- Умови та положення (якщо є)

ДОДАЙ:
- Примітку "Кошторис Q2026-042 від [дата] прийнято клієнтом [дата прийняття]"
- Умови оплати: 30 календарних днів
- Штрафні санкції: [згідно з договором]
- Спосіб оплати: [Банківський переказ / Карта / Інше]
- Банківські реквізити (IBAN, назва банку)

ВИДАЛИ (якщо є):
- Напис "Кошторис не є зобов'язанням"
- Термін дії кошторису (більше не актуально для інвойсу)

Формат: PDF
Збережи у: ~/Cowork-Workspace/output/invoices/INV-2026-085.pdf
```

### Крок 3: Перевірка відповідності

Cowork покаже план. **Перевірте**:

- ✓ Чи ідентичні суми в кошторисі та інвойсі.
- ✓ Правильність та унікальність нумерації.
- ✓ Коректність дат (дата інвойсу ≥ дати кошторису).
- ✓ Наявність усіх юридичних реквізитів.

Якщо в замовленні відбулися зміни порівняно з кошторисом:

```
УВАГА: Клієнт вніс зміни після затвердження кошторису

Оригінальний кошторис Q2026-042:
- Послуга А: 5000 грн

Запит клієнта:
- Послуга А: 6000 грн (додано преміум опцію)

Інвойс INV-2026-085:
- Послуга А преміум: 6000 грн
- Додай примітку: "Зміни узгоджено з клієнтом [дата]"
```

### Крок 4: Генерація та архівування

```
Продовжуй конвертацію (Proceed)
```

Перевірте та заархівуйте:

```bash
# Відкрийте згенерований інвойс
open ~/Cowork-Workspace/output/invoices/INV-2026-085.pdf

# Створіть зв'язок між кошторисом та інвойсом для звітності
echo "Кошторис Q2026-042 → Інвойс INV-2026-085 від 20.01.2026" >> ~/Cowork-Workspace/output/correspondences.txt
```

## Приклади промптів

### Проста конвертація

```
Кошторис Q2026-038 прийнято → Створи інвойс INV-2026-074

Кошторис: ~/Cowork-Workspace/input/Q2026-038.pdf
Дата інвойсу: 20.01.2026
Термін оплати: +30 днів (19.02.2026)
Змін у послугах немає

Додай:
- Примітку "Кошторис Q2026-038 від 12.01.2026 прийнято клієнтом 18.01.2026"
- Банківські реквізити: IBAN... МФО...
- Оплата банківським переказом

Збережи у: ~/Cowork-Workspace/output/invoices/INV-2026-074.pdf
```

### Масова конвертація

```
Пакетно перетвори 5 прийнятих кошторисів у ~/Cowork-Workspace/input/accepted-quotes/

Для кожного кошторису:
1. Прочитай номер кошторису Q2026-XXX.
2. Згенеруй інвойс INV-2026-YYY (починаючи нумерацію з INV-2026-090).
3. Дата інвойсу: 20.01.2026.
4. Термін оплати: +30 днів.
5. Додай стандартні реквізити.

Також створи Excel файл-звіт:
- Колонки: Кошторис | Інвойс | Клієнт | Сума | Дата | Термін оплати
- Збережи у: ~/Cowork-Workspace/output/conversions-january-2026.xlsx
```

## Найкращі практики

1. **Зберігайте зв'язок** — Архівуйте кошторис РАЗОМ з інвойсом для простежуваності.
2. **Формальне прийняття** — Завжди майте доказ прийняття пропозиції (лист або підпис).
3. **Розумні терміни** — Виставляйте інвойс не пізніше ніж через 1 місяць після прийняття кошторису.
4. **Унікальна нумерація** — Ніколи не створюйте два інвойси з однаковим номером.
5. **Подвійна перевірка** — Помилка в інвойсі вимагає створення розрахунку-коригування та нового інвойсу, тому краще перевірити одразу.
