# Воркфлоу: Планування Проєктів та Будівництва

> **Орієнтовний час**: 30 хвилин
> **Складність**: Середній
> **Категорія**: 🔧 Виробництво
> **Професії**: Підрядники, Власники бізнесу, Виконроби

🌐 **Мови**: [English](project-planning.en.md) | [Français](project-planning.md) | [Українська 🇺🇦](project-planning.uk.md)

---

## Кейс використання

Вам потрібно організувати складний проєкт (наприклад, ремонт, будівництво або запуск виробництва) з чіткими термінами, етапами та координацією багатьох учасників. Цей воркфлоу допоможе вам:

- Створити детальний графік робіт (діаграму Ганта).
- Спланувати закупівлі матеріалів відповідно до етапів.
- Координувати роботу різних підрядників/бригад.
- Відстежувати критичний шлях та запобігати затримкам.

> ⚠️ **Ефект**: Якісне планування зменшує середні затримки на 30% та запобігає перевищенню бюджету на 20%.

## Попередні вимоги

- [ ] Cowork увімкнено в Claude Desktop.
- [ ] Опис проєкту (тип, обсяг, бюджет).
- [ ] Список основних етапів.
- [ ] Бажані дати початку та завершення.
- [ ] Створена робоча папка.

## Покрокова інструкція

### Крок 1: Визначення фаз проєкту

```bash
mkdir -p ~/Cowork-Workspace/planning/[назва-проєкту]
```

Встановіть структуру проєкту:

```
Створи план для проєкту: [Назва проєкту]

ЗАГАЛЬНІ ДАНІ:
- Тип: [Ремонт офісу / Будівництво приватного будинку / тощо]
- Обсяг: [Площа в м2, складність]
- Бюджет: [Сума] грн.
- Дата початку: [Дата]
- Бажана дата завершення: [Дата]

ЕТАПИ (приклад для будівництва):

ФАЗА 1: ПІДГОТОВЧА (Тривалість: [X] тижнів)
Завдання:
- Отримання дозволів (ДІАМ, Дія).
- Облаштування будмайданчика (огорожа, тимчасове світло).
- Демонтаж/розчищення території.
Залежності: Немає.
Ресурси: [Юрист, бригада демонтажу].

ФАЗА 2: ФУНДАМЕНТ ТА КАРКАС (Тривалість: [X] тижнів)
Завдання:
- Земляні роботи.
- Армування та заливка фундаменту.
- Зведення стін/каркаса.
- Набір міцності бетону.
Залежності: Завершення Фази 1.
Критичний шлях: Так.

ФАЗА 3: ІНЖЕНЕРНІ МЕРЕЖІ (Тривалість: [X] тижнів)
Завдання:
- Електромонтаж (чорновий).
- Сантехніка та каналізація.
- Опалення та вентиляція.
Залежності: Фаза 2 завершена.
Ресурси: [Електрик, сантехнік].

[Продовжіть для оздоблення та завершення...]

Збережи етапи у: ~/Cowork-Workspace/planning/[назва]/phases.txt
```

### Крок 2: Створення детального графіка (Ганта)

```
Згенеруй діаграму Ганта для проєкту:

ФОРМАТ ГАНТА:
Тиждень 1-2: ██████ Фаза 1: Підготовка
Тиждень 3-6: ██████████ Фаза 2: Фундамент
Тиждень 7-10: ████████ Фаза 3: Мережі
  Т7-8: ████ Електромонтаж
  Т8-9: ████ Сантехніка
  Т9-10: ██ Технічний нагляд/Перевірка

КЛЮЧОВІ ВІХИ (Milestones - моменти оплати/перевірки):
⭐ Тиждень 2: Майданчик готовий → Аванс за матеріали.
⭐ Тиждень 6: Фундамент прийнятий → Оплата за етап.
⭐ Тиждень 10: Мережі перевірені (акти прихованих робіт) → Початок оздоблення.

ЗАЛЕЖНОСТІ:
→ Фундамент МАЄ бути готовий до початку мереж.
→ Перевірка мереж МАЄ пройти до закриття стін гіпсокартоном.

Збережи у: ~/Cowork-Workspace/planning/[назва]/gantt-schedule.xlsx
```

### Крок 3: Планування закупівель матеріалів

Координуйте доставку, щоб не захаращувати об'єкт, але й не простоювати:

```
Склади графік закупівель для проєкту [Назва]:

ЛОГІКА ЗАМОВЛЕННЯ:
Дата замовлення = Дата монтажу - (Термін доставки + Буфер 2 дні)

ФАЗА 2: ФУНДАМЕНТ
Матеріали:
- Бетон М300: [Об'єм] (доставка міксером).
- Арматура 12мм: [Вага].
Термін замовлення: Тиждень 2 (доставка на Тиждень 3).
Постачальник: [Назва/Прайс].

ФАЗА 5: ОЗДОБЛЕННЯ
Матеріали:
- Плитка: [Площа] (Термін доставки: 4 тижні - КРИТИЧНО).
- Фарба: [К-сть].
Застереження: Плитка має бути замовлена на 12-му тижні!

Збережи у: ~/Cowork-Workspace/planning/[назва]/material-procurement.xlsx
```

### Крок 4: Координація підрядників

```
Склади список підрядників та графік виходу:

БРИГАДА 1: ЗАГАЛЬНОБУДІВЕЛЬНА
- Обсяг: Фундамент, стіни.
- Термін: Тиждень 3-6.
- Контакт: [Ім'я, телефон].

БРИГАДА 2: ЕЛЕКТРИКИ
- Обсяг: Чорнова та чистова електрика.
- Важливо: Мають вийти одночасно з сантехніками для координації в санвузлах.

Збережи у: ~/Cowork-Workspace/planning/[назва]/contractor-list.xlsx
```

### Крок 5: Відстеження прогресу (Weekly Report)

```
Згенеруй шаблон щотижневого звіту:

ЗВІТ ЗА ТИЖДЕНЬ [№]
1. ПЛАН: Що мали зробити.
2. ФАКТ: Що зроблено фактично.
3. ВІДХИЛЕННЯ: Чому затримка (погода, логістика).
4. ПЛАН НА НАСТУПНИЙ ТИЖДЕНЬ.
5. БЮДЖЕТ: Витрачено / Залишок.

Збережи у: ~/Cowork-Workspace/planning/[назва]/weekly-report-template.pdf
```

## Найкращі практики

1. **Критичний шлях (Critical Path)** — Визначте завдання, затримка яких зупиняє весь проєкт. Приділяйте їм 80% уваги.
2. **Буферний час** — Завжди закладайте +10-15% часу на непередбачувані обставини (негода, хвороби, затримки логістики).
3. **Акти прихованих робіт** — Завжди фіксуйте на фото та папері мережі (електрика, труби) до того, як вони будуть закриті бетоном чи штукатуркою.
4. **Раннє замовлення** — Матеріали з довгим терміном доставки (індивідуальні вікна, плитка, двері) замовляйте в першу чергу.
5. **Дотримання ДБН/ДСТУ** — Вказуйте підрядникам на необхідність дотримання державних норм, це полегшить введення в експлуатацію.
