## 💡 Загальні поради

### 1. Будьте точними в запитах
```
❌ "Створи інвойс"
✅ "Створи інвойс для [Назва Клієнта], послуги [Назва] 
   від [Дата], сума [Сума], оплата протягом 30 днів"
```

### 2. Завжди перевіряйте результати
Claude потужний, але не безпомилковий:
- Перевіряйте розрахунки (податки, підсумки).
- Перевіряйте реєстраційні дані та адреси.
- Контролюйте логіку даних.
