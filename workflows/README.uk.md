## ⚡ Як використовувати ці воркфлоу?

1. **Виберіть воркфлоу** зі списку вище.
2. **Скопіюйте промпт** (запит), який знаходиться в розділі "Prompt".
3. **Надайте контекст**: Підготуйте необхідні файли у вашій робочій папці Cowork.
4. **Запустіть у Claude**: Вставте промпт, перевірте план дій та натисніть "Run".
