## 🚀 Швидкий старт

### Перед початком воркфлоу

**1. Загальні вимоги:**
```bash
# Створіть робочий простір, якщо потрібно
mkdir -p ~/Cowork-Workspace/{input,output}

# Рекомендовано: зробіть бекап важливих файлів
cp -R ~/Cowork-Workspace/ ~/Cowork-Backup-$(date +%Y%m%d)/
```

**2. Чек-лист запуску:**
- [ ] Cowork увімкнено в Claude Desktop.
- [ ] Робочу папку створено та надано доступ.
- [ ] Вихідні файли покладено в `input/` (якщо потрібно).
- [ ] Ціль воркфлоу чітко визначена.

### Структура кожного воркфлоу

1. **Кейс використання** — Коли застосовувати.
2. **Попередні вимоги** — Що вам знадобиться.
3. **Покрокова інструкція** — Детальна процедура.
4. **Приклади промптів** — Готові команди.
5. **Усунення несправностей** — Поширені проблеми.
6. **Варіації** — Адаптація під контекст.
7. **Найкращі практики** — Професійні поради.

## 💡 Загальні поради

### 1. Будьте точними в запитах

```
❌ "Створи інвойс"
✅ "Створи інвойс для ТОВ 'Клієнт', послуги сантехніка
   від 15.03, сума 8500 грн + ПДВ, оплата протягом 30 днів"
```

### 2. Тестуйте на малих обсягах

Перш ніж обробляти 100 інвойсів, протестуйте воркфлоу на 3-5 прикладах.

### 3. Завжди перевіряйте результати

Claude потужний, але не безпомилковий:
- Перевіряйте розрахунки (ПДВ, підсумки).
- Перевіряйте юридичні дані (ЄДРПОУ, адреси).
- Контролюйте логіку даних.
