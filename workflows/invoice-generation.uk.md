# Воркфлоу: Генерація інвойсів

> **Орієнтовний час**: 15 хвилин
> **Складність**: Новачок
> **Категорія**: 📋 Адміністративні
> **Професії**: Підрядники, Торговці, Фрілансери, Малий бізнес

🌐 **Мови**: [English](invoice-generation.en.md) | [Français](invoice-generation.md) | [Українська 🇺🇦](invoice-generation.uk.md)

---

## Кейс використання

Вам потрібно швидко створити професійні рахунки на основі шаблонів або існуючих кошторисів. Цей воркфлоу допоможе вам:

- Генерувати інвойси відповідно до стандартів за допомогою шаблонів.
- Перетворювати прийняті кошториси в інвойси.
- Автоматично нумерувати рахунки.
- Автоматично розраховувати податки (ПДВ) та підсумкові суми.

> ⚠️ **Важливо**: Цей воркфлоу генерує інвойси у текстовому/PDF форматі. Завжди перевіряйте суми та юридичні реквізити перед відправкою клієнту.

---

## Попередні вимоги

- [ ] Cowork увімкнено в Claude Desktop.
- [ ] Шаблон інвойсу (Word, Excel або PDF) АБО дані клієнта.
- [ ] Наступний номер інвойсу.
- [ ] Створена робоча папка.

---

## Покрокова інструкція

### Крок 1: Підготуйте робочий простір

```bash
# Створіть структуру папок
mkdir -p ~/Cowork-Workspace/{templates,output/invoices}

# Якщо у вас є шаблон, скопіюйте його
cp ~/Documents/invoice-template.docx ~/Cowork-Workspace/templates/
```

### Крок 2: Створення інвойсу за шаблоном

Якщо у вас є готовий шаблон:

```
Використовуй шаблон у ~/Cowork-Workspace/templates/invoice-template.docx

Заповни його наступною інформацією:
- Номер інвойсу: INV-2026-001
- Дата: 20 січня 2026 року
- Клієнт: [Назва та адреса клієнта]
- Послуги:
  * [Опис 1]: [кількість] x [ціна за одиницю]
  * [Опис 2]: [кількість] x [ціна за одиницю]
- Умови оплати: 30 днів

Розрахуй автоматично:
- Проміжний підсумок (Subtotal)
- Податок (ПДВ 20%)
- Разом до оплати

Збережи як ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf
```

### Крок 3: Створення інвойсу з нуля

Якщо у вас немає шаблону:

```
Створи професійний інвойс із наступними даними:

ШАПКА:
- Мої реквізити: [ПІБ/Назва, адреса, ЄДРПОУ/ІПН]
- Логотип: (опціонально)

КЛІЄНТ:
- Назва: [Назва клієнта]
- Адреса: [Повна адреса]

ІНВОЙС:
- Номер: INV-2026-001
- Дата виписки: 20.01.2026
- Термін оплати: 19.02.2026

ПОСЛУГИ:
| Опис | К-сть | Ціна за од. | Разом |
|-------------|-----|------------|-------|
| [Послуга 1] | 1   | 5000.00 грн | 5000.00 грн |
| [Послуга 2] | 2   | 2500.00 грн | 5000.00 грн |

ПІДСУМКИ:
- Сума без ПДВ: 10000.00 грн
- ПДВ (20%): 2000.00 грн
- Разом до оплати: 12000.00 грн

ЮРИДИЧНІ ПРИМІТКИ:
- Умови оплати: протягом 30 днів
- Штраф за прострочення: [згідно з договором/законом]

Формат: PDF
Збережи у: ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf
```

### Крок 4: Перевірка інвойсу

Cowork покаже свій план. **Перевірте**:

- ✓ Правильність розрахунків (підсумок, ПДВ, разом).
- ✓ Унікальність номера інвойсу.
- ✓ Повноту контактних даних.
- ✓ Наявність обов'язкових юридичних реквізитів.

### Крок 5: Генерація та збереження

```
Продовжуй генерацію (Proceed)
```

Після генерації:

```bash
# Перегляньте файл (macOS)
open ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf

# Або в Linux
xdg-open ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf
```

## Приклади промптів

### Простий інвойс

```
Створи інвойс:
- №: INV-2026-042
- Дата: 20.01.2026
- Клієнт: ТОВ "Меблі Плюс", вул. Дубова 123, Київ, 01001
- Послуга: Встановлення вікон x3, 4500 грн кожне
- ПДВ 20%
- Умови оплати: 30 днів

Мої реквізити:
ФОП Іванов, вул. Головна 456, Київ, 01002
ЄДРПОУ: 12345678

Збережи як PDF: ~/Cowork-Workspace/output/invoices/INV-2026-042.pdf
```

### Масове створення інвойсів

```
Створи 5 інвойсів на основі кошторисів у ~/Cowork-Workspace/input/

Для кожного прийнятого кошторису:
- Перетвори на інвойс.
- Нумерація: INV-2026-[порядковий номер].
- Збережи ті самі суми.
- Додай термін оплати: +30 днів.
- Формат PDF у папку ~/Cowork-Workspace/output/invoices/.
```

## Усунення несправностей

### Неправильний розрахунок податків

**Причина**: Ставка податку не вказана або вказана неправильно.
**Рішення**: Явно вкажіть ставку:

```
Застосуй податок: 20% (основна ставка ПДВ)
АБО
Застосуй податок: 7% (пільгова ставка)
АБО
Без ПДВ (вкажіть причину звільнення)
```

### Дублювання номерів

**Причина**: Номер інвойсу вже використовувався.
**Рішення**: Ведіть реєстр вручну або запитайте у Cowork:

```
Перелічи всі інвойси в ~/Cowork-Workspace/output/invoices/
Назви мені наступний вільний номер
```

### Відсутні юридичні реквізити

**Причина**: Неповний шаблон або промпт.
**Рішення**: Надайте всі обов'язкові дані:

```
Додай ці реквізити:
- Мій ЄДРПОУ: [номер]
- Реєстраційні дані: [якщо потрібно]
- Умови оплати
- Пеня за прострочення: [ставка]
```

## Варіації

### Рахунок на аванс (депозит)

```
Згенеруй рахунок на аванс:
- Сума: 30% від кошторису Q2026-015
- Примітка: "Передоплата 30% - згідно з кошторисом Q2026-015"
- Остаточний інвойс буде за вирахуванням цього авансу
```

### Регулярний (щомісячний) рахунок

```
Створи щомісячний інвойс за січень 2026:
- Той самий клієнт, що в попередньому рахунку INV-2026-035
- Та сама послуга: Щомісячне обслуговування
- Та сама сума: 2500 грн
- Нова дата і номер: INV-2026-045
```

## Найкращі практики

1. **Послідовна нумерація** — Ніяких пропусків у номерах (це законодавча вимога в багатьох юрисдикціях).
2. **Систематичне резервне копіювання** — Зберігайте всі інвойси протягом встановленого законом терміну (зазвичай 3-10 років).
3. **Подвійна перевірка** — Завжди переглядайте файл перед відправкою клієнту.
4. **Валідований шаблон** — Нехай ваш бухгалтер перевірить шаблон на відповідність чинному законодавству України.
5. **Організований архів** — Використовуйте зрозумілі назви файлів (рік/номер або назва клієнта).
