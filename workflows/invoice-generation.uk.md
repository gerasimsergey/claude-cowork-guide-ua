## Кейс використання

Вам потрібно швидко створити професійні інвойси (рахунки-фактури) за допомогою шаблонів або на основі існуючих кошторисів. Цей воркфлоу допоможе вам:

- Генерувати інвойси за шаблонами
- Перетворювати прийняті кошториси на інвойси
- Автоматично нумерувати інвойси
- Автоматично розраховувати податки та підсумки

> **Важливо**: Цей воркфлоу генерує інвойси у текстовому або PDF форматі. Завжди перевіряйте суми та юридичні реквізити перед надсиланням.

## Покрокова інструкція

### Крок 1: Підготовка робочого простору

```bash
# Створення структури
mkdir -p ~/Cowork-Workspace/{templates,output/invoices}

# Якщо у вас є шаблон, скопіюйте його
cp ~/Documents/invoice-template.docx ~/Cowork-Workspace/templates/
```

### Крок 2: Створення інвойсу за шаблоном

Якщо у вас є шаблон:

```
Використовуй шаблон у ~/Cowork-Workspace/templates/invoice-template.docx

Заповни його такою інформацією:
- Номер інвойсу: INV-2026-001
- Дата: 20 січня 2026 року
- Клієнт: [Назва та адреса клієнта]
- Послуги:
  * [Опис 1]: [кількість] x [ціна за одиницю]
  * [Опис 2]: [кількість] x [ціна за одиницю]
- Умови оплати: Net 30

Розрахуй автоматично:
- Проміжний підсумок (Subtotal)
- Податок (ПДВ/Податок з продажу)
- Разом (Total)

Збережи в ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf
```

### Крок 3: Створення інвойсу з нуля

Якщо у вас немає шаблону:

```
Створи професійний інвойс із такими даними:

ШАПКА:
- Мої дані: [Назва, адреса, ІПН/ЄДРПОУ]
- Логотип: (опціонально)

КЛІЄНТ:
- Назва: [Назва клієнта]
- Адреса: [Повна адреса]

ІНВОЙС:
- Номер: INV-2026-001
- Дата виписки: 20.01.2026
- Термін оплати: 19.02.2026

ПОСЛУГИ:
| Опис | К-ть | Ціна за од. | Разом |
|------|------|-------------|-------|
| [Послуга 1] | 1 | $500.00 | $500.00 |
| [Послуга 2] | 2 | $250.00 | $500.00 |

ПІДСУМКИ:
- Проміжний підсумок: $1,000.00
- Податок (20%): $200.00
- Разом: $1,200.00

ЮРИДИЧНІ ПРИМІТКИ:
- Умови оплати: Net 30 днів
- Пеня за затримку: [законна ставка]
- Знижка за ранню оплату: немає

Формат: PDF
Зберегти до: ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf
```

### Крок 4: Перевірка інвойсу

Cowork відобразить свій план. **Перевірте**:

- ✓ Правильність розрахунків (підсумок, податки, разом)
- ✓ Унікальність номера інвойсу
- ✓ Повноту контактних даних
- ✓ Наявність необхідних юридичних приміток

### Крок 5: Генерація та збереження

```
Продовжуй генерацію
```

Після генерації:

```bash
# Перевірка файлу (macOS)
open ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf

# АБО на Linux
xdg-open ~/Cowork-Workspace/output/invoices/INV-2026-001.pdf
```

```
- Умови оплати: Net 30

Зберегти як PDF: ~/Cowork-Workspace/output/invoices/INV-2026-042.pdf
```

### Пакетна генерація інвойсів

```

```

## Усунення несправностей

### Неправильний розрахунок податків

**Причина**: Податкова ставка не вказана або вказана неправильно
**Рішення**: Чітко вкажіть її:

```
Застосовний податок: 20% (стандартна ставка)
АБО
Застосовний податок: 10% (знижена ставка)
АБО
Без ПДВ (вкажіть причину звільнення)
```

### Дублювання номерів

**Причина**: Номер інвойсу вже використовувався
**Рішення**: Ведіть журнал вручну або запитайте:

```
Перелічи всі інвойси в ~/Cowork-Workspace/output/invoices/
Дай мені наступний доступний номер
```

### Відсутність юридичних приміток

**Причина**: Неповний шаблон
**Рішення**: Надайте всі необхідні примітки:

```
Додай ці примітки:
- ІПН/ЄДРПОУ: [ваш номер]
- Реєстрація компанії: [якщо актуально]
- Умови оплати
- Пеня за несвоєчасну оплату: [ставка]
- Витрати на стягнення: [сума]
```

## Варіації

### Інвойс на депозит (аванс)

```
Згенеруй інвойс на депозит:
- Сума: 30% від кошторису Q2026-015
- Примітка: "Депозит 30% - Кошторис Q2026-015"
- Фінальний інвойс буде виписаний за вирахуванням цього депозиту
```

### Регулярний інвойс (щомісячний)

```
Створи щомісячний інвойс за січень 2026 року:
- Той самий клієнт, що і в попередньому інвойсі INV-2026-035
- Та сама послуга: Щомісячне обслуговування
- Та сама сума: $250
- Нова дата та номер: INV-2026-045
```

### Експорт в Excel для бухгалтерії

```
Після генерації PDF-інвойсу також створи:
- Файл Excel: ~/Cowork-Workspace/output/invoices/INV-2026-042.xlsx
- Зі стовпцями: Дата, №, Клієнт, Підсумок, Податок, Разом, Термін оплати
```

## Найкращі практики

1. **Послідовна нумерація** — Відсутність пропусків у нумерації (законна вимога в багатьох країнах)
2. **Систематичне резервне копіювання** — Зберігайте всі інвойси протягом необхідного терміну зберігання (зазвичай 7-10 років)
3. **Подвійна перевірка** — Завжди переглядайте документ перед надсиланням клієнту
4. **Затверджений шаблон** — Погодьте ваш шаблон з бухгалтером
5. **Оновлені юридичні примітки** — Щорічно перевіряйте законодавчі вимоги до оформлення документів
6. **Організоване архівування** — Послідовне іменування файлів (рік/номер або за клієнтом)
