# Воркфлоу: Дослідження Потенційних Клієнтів

> **Орієнтовний час**: 20 хвилин
> **Складність**: Середній
> **Категорія**: 💼 Комерційні
> **Професії**: Торговці, Власники бізнесу, Менеджери з продажу

🌐 **Мови**: [English](prospect-research.en.md) | [Français](prospect-research.md) | [Українська 🇺🇦](prospect-research.uk.md)

## Кейс використання

У вас є потенційний клієнт (лід), і вам потрібно зібрати максимум інформації перед зустріччю або підготовкою комерційної пропозиції. Цей воркфлоу допоможе вам:

- Зібрати юридичні та фінансові дані компанії (діяльність, розмір, стан).
- Ідентифікувати осіб, які приймають рішення (ОПР).
- Зрозуміти потенційні потреби клієнта через його цифрову присутність.
- Підготувати персоналізований підхід для "теплого" контакту.

> ⚠️ **Ефективність**: Добре досліджені контакти дають на 40% вищу конверсію порівняно з "холодними" дзвінками навмання.

## Покрокова інструкція

### Крок 1: Юридична інформація про компанію

```bash
mkdir -p ~/Cowork-Workspace/prospects/[назва-компанії]
```

Офіційне дослідження (Україна):

```
Збери юридичну інформацію для: [Назва компанії / Код ЄДРПОУ]

ПУБЛІЧНІ ДЖЕРЕЛА:
1. Опендатабот / YouControl / Clarity Project:
   - Повна юридична назва.
   - Статус (активне/в процесі припинення).
   - КВЕДи (основний та додаткові).
   - ПІБ засновників та директора.
   - Розмір статутного капіталу.
   - Наявність податкового боргу.
   - Судові справи (якщо є).

2. Фінансові показники (якщо доступні):
   - Дохід за останній рік.
   - Динаміка зростання/падіння.

Збережи дані у: ~/Cowork-Workspace/prospects/[назва]/legal-info.txt
```

### Крок 2: Цифрова присутність та сигнали до покупки

Проаналізуйте, як компанія спілкується онлайн:

```
Проаналізуй веб-присутність для: [Назва компанії]

САЙТ ТА СОЦМЕРЕЖІ:
- Дизайн сайту: сучасний чи застарілий?
- Чи є розділ "Кар'єра"? (Великий набір персоналу = сигнал про зростання та бюджет).
- Останні новини: нові філії, проєкти, нагороди.
- Відгуки клієнтів (Google Maps, Facebook): на що скаржаться? (Це ваша можливість запропонувати краще рішення).

СИГНАЛИ ДО ПОКУПКИ (Buying Signals):
- Перемоги в тендерах (Prozorro).
- Відкриття нових об'єктів/офісів.
- Зміна керівництва.

Збережи у: ~/Cowork-Workspace/prospects/[назва]/digital-presence.txt
```

### Крок 3: Особи, які приймають рішення (ОПР)

Ідентифікуйте стейкголдерів:

```
Знайди ключових осіб у [Назва компанії] через LinkedIn:

- Директор (CEO/Owner).
- Керівник відділу (закупівель, ІТ, маркетингу - залежно від вашої послуги).
- Операційні менеджери.

Для кожної особи знайди:
- ПІБ.
- Посада.
- Досвід (як довго в компанії).
- Останні публікації (про що людина пише).

Сформуй ієрархію прийняття рішень (Decision Map).
Збережи у: ~/Cowork-Workspace/prospects/[назва]/contacts.txt
```

### Крок 4: Кваліфікація та скоринг

Оцініть потенціал ліда:

```
На основі зібраних даних оціни потенціал клієнта [Назва] для моїх послуг [Опис ваших послуг] за шкалою від 1 до 100:

КРИТЕРІЇ:
- Відповідність розміру компанії моєму профілю (/20).
- Наявність бюджету (фінансові сигнали) (/20).
- Чітко виявлена потреба (/30).
- Доступність ОПР (/15).
- Таймінг (терміновість) (/15).

ПІДСУМОК:
> 70: Гарячий лід (контактувати негайно).
50-70: Теплий лід (потребує прогріву).
< 50: Холодний лід (додати в розсилку).

Збережи у: ~/Cowork-Workspace/prospects/[назва]/scoring.txt
```

### Крок 5: Консолідоване досьє клієнта

```
Згенеруй фінальну картку потенційного клієнта (Prospect Sheet):

1. ЗАГАЛЬНА ІНФОРМАЦІЯ (Діяльність, дохід, штат).
2. КЛЮЧОВІ ОПР та їх контакти.
3. ВИЯВЛЕНІ ПОТРЕБИ (На основі аналізу болів та сигналів).
4. РЕКОМЕНДОВАНИЙ ГАЧОК (З чого почати розмову).
5. МОЖЛИВІ ЗАПЕРЕЧЕННЯ (До чого готуватися).

Формат: PDF (1 сторінка).
Збережи у: ~/Cowork-Workspace/prospects/[назва]/PROSPECT-CARD.pdf
```

## Найкращі практики

1. **Кваліфікуйте ПЕРЕД контактом** — 15 хвилин дослідження заощаджують 2 години марних переговорів.
2. **Використовуйте YouControl/Opendatabot** — Для українського ринку це найнадійніші джерела юридичної чистоти бізнесу.
3. **Шукайте "точки болю"** — Якщо на Google Maps скаржаться на повільну доставку, а ви пропонуєте логістику — це ваш ідеальний "гачок".
4. **LinkedIn — це золото** — Дивіться не тільки на посаду, а й на те, що людина лайкає чи коментує.
5. **Дотримуйтесь етики** — Збирайте тільки публічно доступну професійну інформацію.
