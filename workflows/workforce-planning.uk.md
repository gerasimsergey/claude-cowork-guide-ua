# Воркфлоу: Планування Персоналу (HR)

🌐 **Мови**: [English](workforce-planning.en.md) | [Français](workforce-planning.md) | [Українська 🇺🇦](workforce-planning.uk.md)

> **Орієнтовний час**: 40-60 хвилин
> **Складність**: Середній
> **Категорія**: 📊 Адміністративні
> **Професії**: HR-менеджери, Операційні директори, Власники бізнесу

---

## Кейс використання

Вам потрібно спланувати штат на наступний квартал або рік і ви хочете:

- Консолідувати поточну структуру команди з різних джерел.
- Змоделювати сценарії зростання (найм, звільнення, підвищення).
- Розрахувати загальні витрати на оплату праці (Total Compensation).
- Візуалізувати зміни в організаційній структурі.
- Підготувати обґрунтування бюджету для зацікавлених сторін.

> ⚠️ **Важливо**: Цей воркфлоу використовує **можливості Cowork з генерації Excel** (створення нових файлів планування). Він НЕ використовує **надбудову Claude в Excel**. Див. [порівняння](../reference/comparison.uk.md#common-confusion).

---

## Попередні вимоги

- [ ] Cowork увімкнено (підписка Pro або Max).
- [ ] Поточний список співробітників (CSV, Excel або текст).
- [ ] Дані про зарплати та податки.
- [ ] Цілі зростання або нотатки щодо плану найму.
- [ ] Створена робоча папка.

## Покрокова інструкція

### Крок 1: Підготуйте вхідні дані

```bash
# Створіть структуру робочого простору
mkdir -p ~/Cowork-Workspace/{input/hr-data,output}

# Скопіюйте дані про персонал
cp ~/шлях/до/employee-list.xlsx ~/Cowork-Workspace/input/hr-data/
```

**Порада**: Анонімізуйте конфіденційні дані (замініть імена на ID), якщо ви плануєте показувати модель широкому колу осіб.

### Крок 2: Консолідація поточної чисельності (Baseline)

Створіть базовий зріз команди:

```
Проаналізуй дані про співробітників у ~/Cowork-Workspace/input/hr-data/

Створи ~/Cowork-Workspace/output/headcount-baseline.xlsx із листами:

Лист 1 - "Поточна команда":
- ID співробітника
- Департамент
- Посада
- Тип зайнятості (Full-time, Part-time, ФОП, Контракт)
- Дата виходу на роботу
- Оклад (Gross)
- Податки (ЄСВ, ПДФО, ВЗ - розрахуй за ставками України)
- Загальні витрати компанії

Лист 2 - "Підсумок за відділами":
- Департамент
- Штатна чисельність (FTE)
- Загальний фонд оплати праці (ФОП)
- Середня зарплата по відділу
```

### Крок 3: Моделювання сценаріїв зростання

Планування найму та плинності:

```
Використовуючи дані з ~/Cowork-Workspace/output/headcount-baseline.xlsx

Створи ~/Cowork-Workspace/output/growth-scenarios.xlsx:

Лист 1 - "План на 2026 рік":
- Поточна чисельність (за відділами)
- Запланований найм (посада, місяць виходу, очікувана зарплата)
- Прогнозована плинність (звільнення)
- Чиста зміна штату
- Вплив на загальний бюджет

Лист 2 - "Прогноз витрат":
- Місяць (Січень-Грудень 2026)
- Витрати на основний штат
- Витрати на нових співробітників
- Податки на фонд оплати праці
- Загальні місячні витрати
- Накопичувальні витрати за рік
```

### Крок 4: Аналіз впливу ролей

Визначте пріоритети найму:

```
На основі сценаріїв зростання в ~/Cowork-Workspace/output/growth-scenarios.xlsx

Створи ~/Cowork-Workspace/output/role-impact.xlsx:
- Пріоритетні ролі (за впливом на бізнес: Високий/Середній/Низький)
- Необхідні навички для зростання
- Аналіз прогалин (Skills Gap): що ми маємо vs що нам потрібно
- Рекомендація: Навчати наявних чи наймати нових?
```

### Крок 5: Візуалізація структури (Org Chart)

```
На основі поточної бази та плану найму:

Створи документ Word ~/Cowork-Workspace/output/org-chart.docx:
1. Поточна оргструктура (ієрархічний список).
2. Пропонована структура на кінець року (з новими ролями).
3. Виділи зміни: НОВИЙ, ЗВІЛЬНЕННЯ, ПІДВИЩЕННЯ.
```

### Крок 6: Звіт для керівництва (Budget Justification)

```
Створи документ Word ~/Cowork-Workspace/output/workforce-plan-2026.docx

Включи:
1. Резюме (Executive Summary): Поточний штат vs Цільовий, Загальний бюджет, % зростання.
2. Стратегічне обґрунтування: Чому цей найм критичний для цілей бізнесу.
3. Фінансовий вплив: Графік зростання витрат по місяцях.
4. Ризики: Що буде, якщо найм буде затримано.
```

## Специфіка України (Податки)

При розрахунках в Excel використовуйте такі параметри для України:
- **ПДФО**: 18%
- **Військовий збір**: 1.5% (або актуальна ставка)
- **ЄСВ**: 22% (нараховується на фонд оплати праці)
- **ФОП (3 група)**: 5% + ЄСВ (якщо використовуєте таку модель співпраці)

**Приклад доповнення до промпту**:
```
Враховуй податкове навантаження України:
- Для штатних працівників: Нарахована сума (Gross) + 22% ЄСВ = Загальні витрати компанії.
- Для співпраці з ФОП: Сума контракту + ЄСВ = Загальні витрати.
```

## Найкращі практики

1. **Анонімність** — При роботі з ШІ краще використовувати ID співробітників замість ПІБ для збереження приватності.
2. **Три сценарії** — Завжди моделюйте три варіанти: Консервативний (мінімум найму), Базовий (плановий) та Агресивний (швидке зростання).
3. **Плинність кадрів** — Закладайте в план середній відсоток плинності (наприклад, 10-15% на рік), щоб бюджет був реалістичним.
4. **Формули в Excel** — Просіть Cowork створювати файли з формулами, а не просто значеннями, щоб ви могли самі змінювати дати найму або зарплати.
