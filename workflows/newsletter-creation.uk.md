# Воркфлоу: Створення Розсилок (Newsletters)

> **Орієнтовний час**: 25 хвилин
> **Складність**: Середній
> **Категорія**: 📣 Комунікація
> **Професії**: Торговці, Власники бізнесу, Маркетологи

🌐 **Мови**: [English](newsletter-creation.en.md) | [Français](newsletter-creation.md) | [Українська 🇺🇦](newsletter-creation.uk.md)

## Кейс використання

Вам потрібно регулярно нагадувати про себе клієнтам, щоб стимулювати повторні продажі та будувати довіру. Цей воркфлоу допоможе вам:

- Сформувати цікаву редакційну лінію (про що писати).
- Скласти контент-план розсилок на квартал.
- Написати тексти листів, які відкривають і читають.
- Аналізувати результати (відкриття, кліки, продажі).

> ⚠️ **Ефективність**: Регулярні розсилки (хоча б 1 раз на місяць) підвищують частоту повторних покупок на 25%.

---

## Попередні вимоги

- [ ] Cowork увімкнено в Claude Desktop.
- [ ] База контактів клієнтів (email), які дали згоду на розсилку.
- [ ] Сервіс для розсилок (напр. SendPulse, eSputnik, Brevo, Mailchimp).
- [ ] Створена робоча папка.

## Покрокова інструкція

### Крок 1: Визначення редакційної політики

```bash
mkdir -p ~/Cowork-Workspace/newsletters
```

Встановіть правила гри:

```
Сформуй редакційну лінію для розсилки: [Назва бізнесу]

ЦІЛЬ: [Навчання клієнтів / Прямі продажі / Новини галузі]
ПЕРІОДИЧНІСТЬ: [Наприклад: 1 раз на 2 тижні]

СТРУКТУРА ЛИСТА (Правило 80/20):
- 80% КОРИСТІ: Поради, кейси, відповіді на питання.
- 20% ПРОДАЖІВ: Акції, нові товари, заклики до дії.

ТОН: [Дружній / Офіційний / Експертний]
Збережи у: ~/Cowork-Workspace/newsletters/editorial-line.txt
```

### Крок 2: Складання контент-плану

```
Склади план розсилок на наступні 3 місяці:

СІЧЕНЬ: Тема "Підготовка до сезону". Порада про [X]. Акція на [Y].
ЛЮТИЙ: Тема "Економія ресурсів". Кейс клієнта.
БЕРЕЗЕНЬ: Новинки весни.

Збережи у: ~/Cowork-Workspace/newsletters/calendar.xlsx
```

### Крок 3: Написання конкретного листа

```
Напиши текст розсилки на тему: [Тема місяця]

ОБОВ'ЯЗКОВІ ЕЛЕМЕНТИ:
1. Тема листа (Catchy subject line): Створи 3 варіанти, щоб клієнт захотів відкрити лист.
2. Прехедер (Preview text): Коротке пояснення, про що лист.
3. Привітання та "гачок" на початку.
4. Основна корисна частина (3 поради).
5. Блок "Товар місяця" або "Спеціальна пропозиція".
6. Кнопка (CTA): Наприклад, "Дізнатися ціну" або "Читати статтю".
7. Футер: Реквізити компанії та посилання на відписку (обов'язково!).
```

### Крок 4: Оптимізація та перевірка

```
Перевір написаний текст:
- Чи немає "спам-слів", через які лист потрапить у спам (напр. "Безкоштовно!!!", "Мільйон")?
- Чи легко читається текст на мобільному (короткі абзаци)?
- Чи чіткий заклик до дії?
```

### Крок 5: Аналіз ефективності (Post-campaign)

Після відправки вставте дані з сервісу розсилок:

```
Проаналізуй результати розсилки:
- Надіслано: 500
- Відкрито (Open Rate): 25%
- Клікнули (CTR): 5%
- Відписалися: 2

Висновок: Чи була тема вдалою? Що змінити в наступному листі?
```

## Найкращі практики

1. **Дозвіл — це святе** — Ніколи не купуйте бази адрес. Надсилайте листи тільки тим, хто сам залишив контакт. Це гарантує доставку і лояльність.
2. **Персоналізація** — Використовуйте ім'я клієнта в темі або на початку листа. "Олександре, ваші поради на березень" працює краще, ніж "Наші поради".
3. **Один лист — одна ціль** — Не намагайтеся продати все відразу. Оберіть одну головну кнопку, на яку клієнт має натиснути.
4. **Легкість відписки** — Посилання на відписку має бути помітним. Краще, щоб незацікавлений клієнт відписався, ніж поскаржився на спам.
5. **Тестуйте час** — Спробуйте надсилати листи у вівторок о 10:00 або в четвер о 14:00. В Україні це зазвичай пікові години перевірки пошти.
