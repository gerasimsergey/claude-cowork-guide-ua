# Воркфлоу: Створення розсилки

> **Орієнтовний час**: 25 хвилин
> **Складність**: Середня

🌐 **Мови**: [English](newsletter-creation.en.md) | [Français](newsletter-creation.md) | [Українська 🇺🇦](newsletter-creation.uk.md)

## Кейс використання

Вам потрібно створювати регулярну розсилку (newsletter) для підтримки зв'язку з клієнтами та потенційними замовниками. Цей воркфлоу допоможе вам:

- Структурувати ефективний контент для розсилки
- Генерувати тексти, адаптовані під вашу аудиторію
- Планувати редакційний календар
- Вимірювати рівень залученості

> ⚠️ **Ефективність**: Регулярні розсилки (мінімум 1 раз на місяць) забезпечують +25% до рівня повторних покупок порівняно з відсутністю комунікації.

## Покрокова інструкція

### Крок 1: Визначення редакційної політики

```bash
mkdir -p ~/Cowork-Workspace/newsletters
```

Встановлення рамок:

```
Визнач редакційну політику розсилки для: [Назва компанії]

МІЙ БІЗНЕС: [Напр: Електрик, Магазин одягу, Консультант]
МОЯ АУДИТОРІЯ: [B2B клієнти / Споживачі / Змішана]
МЕТА РОЗСИЛКИ: [Утримання / Продажі / Експертність / Змішана]

ЧАСТОТА:
- Щомісяця (рекомендований мінімум)
- Двічі на місяць (якщо багато контенту)
- Щокварталу (якщо мало оновлень)

ТИПОВА СТРУКТУРА (Правило третин):

1/3 НАВЧАЛЬНИЙ КОНТЕНТ (Додана цінність):
- Поради галузі
- Практичні посібники
- Тренди сектору
- Відповіді на поширені запитання

Приклади для електрика:
- "5 ознак того, що ваша електромережа небезпечна"
- "Як зменшити рахунок за світло на 20%"
- "Нові будівельні норми: що змінюється для вас"

2/3 НОВИНИ КОМПАНІЇ (Залучення):
- Нові проєкти (до/після)
- Нові послуги
- Команда (нові працівники, річниці)
- Події (виставки, дні відкритих дверей)

Приклади:
- "Ми оновили електромережу в [клієнт] (фото)"
- "Нова послуга: Безкоштовний енергоаудит до 31 березня"
- "Наша команда росте: вітаємо Луку, помічника електрика"

3/3 ПРОМОЦІЇ/CTA (Конверсія):
- Обмежені в часі пропозиції
- Ексклюзивний промокод для підписників
- Запрошення на подію
- Запит на відгук

Приклади:
- "Тільки для підписників: -10% на виклики цього місяця (код: NEWS10)"
- "Порекомендуйте друга та отримайте знижку $50"

ТОН (Tone of Voice):
- Професійний, але доступний
- Формальний або неформальний залежно від аудиторії
- Людський (уникайте надмірного технічного жаргону)

НАЗВА РОЗСИЛКИ:
[Придумайте влучну назву]
Наприклад: "У центрі напруги" (електрик), "Модний лист" (рітейл), "HR-інсайти" (консультант).

Зберегти: ~/Cowork-Workspace/newsletters/editorial-line.txt
```

### Крок 2: Планування контенту (календар)

```
Створи редакційний календар розсилки на 2026 рік:

СІЧЕНЬ:
Тема: Новорічні цілі / Старт року
Навчальний контент: "Чек-лист електробезпеки на новий рік"
Новини: "Наші вітання + підсумки 2025"
Промоція: "Безкоштовний аудит у січні"

ЛЮТИЙ:
Тема: Підготовка до весни
Контент: "Підготовка саду (освітлення, вуличні розетки)"
Новини: "Нове партнерство з [бренд]"
Промоція: "Безкоштовний розрахунок для вуличних проєктів"

[...Продовжити на 12 місяців]

Формат: Таблиця Excel
Колонки: Місяць | Тема | Навчання | Новини | Промоція | Дата надсилання | Статус
Зберегти: ~/Cowork-Workspace/newsletters/editorial-calendar-2026.xlsx
```

```
Згенеруй розсилку за [Місяць] для: [Назва компанії]

ЗАГОЛОВОК:
НАЗВА: [Назва розсилки]
ВИПУСК: #[X]
ДАТА: [Місяць Рік]

PRE-HEADER (текст попереднього перегляду):
"[Гачок до 50 символів, що резюмує головний зміст]"
Приклад: "5 порад із безпеки + Спеціальна пропозиція цього місяця"

ТЕМА ЛИСТА (критично для відкриття):
Ефективні формули:
- "[Ім'я], ваші 5 порад з електрики на цей місяць"
- "Ваш гід з [тема] + знижка 10% цього місяця"
- "Новинка: послуга [назва] вже доступна"

ТІЛО ЛИСТА:

Привітання:
"Вітаю, [Ім'я],
З Новим 2026 роком! Щоб почати рік правильно, ми підготували практичний посібник з електробезпеки. Також дізнайтеся про нашу нову пропозицію безкоштовного аудиту, що діє весь місяць."

ПОРАДА МІСЯЦЯ: 5 ознак того, що ваша електромережа небезпечна
[Список пунктів з діями]

НАШІ ПРОЄКТИ:
Проєкт місяця: Повна реновація для пані Джонсон
[Опис до/після + фото]

СПЕЦІАЛЬНО ДЛЯ ПІДПИСНИКІВ:
Тільки в січні: БЕЗКОШТОВНИЙ енергоаудит (вартість $150)
[Кнопка CTA: Забронювати]
Код для замовлення: NEWS-JAN26

КОРОТКО ПРО ГОЛОВНЕ:
- Наша команда росте...
- Нове партнерство...
- Наступна виставка...

КОНТАКТИ:
Телефон: (555) 123-4567
Стежте за нами: [Соцмережі]

Зберегти: ~/Cowork-Workspace/newsletters/newsletter-january-2026.html
```

### Крок 4: Надсилання та відстеження

```
Відстеження індикаторів (через 48 годин після надсилання):

РІВЕНЬ ВІДКРИТТЯ (Open Rate):
Ціль: >20% (B2B) / >15% (B2C)
Якщо <10%: Змініть тему листа (недостатньо цікава).

РІВЕНЬ КЛІКІВ (CTR):
Ціль: >3%
Якщо <2%: Контент не залучає або слабкий CTA.

РІВЕНЬ ВІДПИСОК:
Припустимо: <0.5%
Якщо >2%: Занадто висока частота або контент не релевантний.

Таблиця відстеження:
| Випуск | Дата | Надіслано | Відкрито | % | Кліки | CTR | Відписки | Конверсії | Дохід |
|--------|------|-----------|----------|---|-------|-----|----------|-----------|-------|
```

## Найкращі практики

1. **Регулярність** — Однакова частота кожного місяця (прогнозованість).
2. **Цінність понад усе** — 70% корисної інформації / 30% промоції.
3. **Mobile-first** — Перевіряйте відображення на мобільних (там читають 70% користувачів).
4. **Легка відписка** — Видиме посилання (вимога закону + позитивний імідж).
5. **Персоналізація** — Використовуйте ім'я, сегментуйте список, якщо можливо.
6. **Чіткий CTA** — Одна головна мета на один випуск.
7. **Гігієна списку** — Видаляйте неактивні адреси для покращення доставності.
8. **Вимірюйте ROI** — Відстежуйте реальні замовлення, а не лише відкриття.
