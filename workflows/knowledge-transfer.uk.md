# Передача знань (Knowledge Transfer)

🌐 **Мови**: [English](knowledge-transfer.en.md) | [Français](knowledge-transfer.md) | [Українська 🇺🇦](knowledge-transfer.uk.md)

Цей воркфлоу допомагає зберегти експертизу всередині компанії, перетворюючи розрізнену інформацію в головах або папках на структуровану базу знань для нових співробітників або клієнтів.

---

## 📋 Контекст

Ви — єдиний, хто знає, як працює певний процес (наприклад, налаштування сервера або запуск реклами). Ви хочете найняти помічника і вам потрібно швидко створити для нього інструкцію.

## 🛠️ Підготовка

1. Зберіть ваші нотатки, скриншоти, записи чатів або шматки коду, пов'язані з процесом.
2. Надайте Claude доступ до цих матеріалів.

---

## 📝 Промпт (Запит)

Скопіюйте цей запит у Claude:

```text
CONTEXT: I am onboarding a new Junior Admin for MacPlus Software.
TASK: Based on the files in 'server_setup_notes/', create a comprehensive 'Onboarding_Guide.md'.
- Step-by-step instructions for setting up a local dev environment.
- List of common errors and how to fix them.
- Links to internal resources (based on my file list).
- A checklist for the new employee to complete in their first week.
OUTPUT: Create a structured guide with clear headings and code blocks.
CONSTRAINTS: Explain complex concepts in simple terms. Ensure all security warnings are highlighted.
```

---

## ⚙️ План виконання

1. **Систематизація**: Claude знайде логіку в хаотичних нотатках та вибудує послідовність кроків.
2. **Спрощення**: Система перефразує складні технічні моменти так, щоб вони були зрозумілі новачкові.
3. **Візуалізація**: Claude структурує текст за допомогою списків, таблиць та блоків коду для кращого сприйняття.

---

## ✅ Перевірка результату

- Дайте почитати інструкцію людині, яка не знайома з процесом — чи все їй зрозуміло?
- Перевірте актуальність усіх технічних кроків.
- Переконайтеся, що критичні застереження (наприклад, щодо безпеки) добре помітні.

---

## 💡 Поради

- **FAQ для клієнтів**: Ви можете використовувати цей воркфлоу для створення сторінки допомоги (Help Center) на вашому сайті на основі ваших відповідей клієнтам у пошті.
- **Відео-транскрипції**: Ви можете завантажити текст (транскрипт) вашого запису екрана, де ви щось пояснюєте, і Claude зробить з нього ідеальну текстову інструкцію зі скриншотами.
