# Воркфлоу: Передача Знань та Досвіду

> **Орієнтовний час**: 2-4 години (залежно від складності)
> **Складність**: Висока
> **Категорія**: 🗂️ Організація
> **Професії**: Майстри, Власники малого бізнесу, Виконроби

🌐 **Мови**: [English](knowledge-transfer.en.md) | [Français](knowledge-transfer.md) | [Українська 🇺🇦](knowledge-transfer.uk.md)

## Кейс використання

Вам потрібно передати свої знання та професійні секрети новому співробітнику, учню або наступнику. Цей воркфлоу допоможе вам:

- Перетворити ваш багаторічний досвід на чітку документацію.
- Створити структуровані навчальні плани для новачків.
- Зберегти унікальні "фішки" та методи роботи вашої компанії.
- Пришвидшити адаптацію нових людей без вашої постійної присутності.

> ⚠️ **Важливо**: 60% малих ремісничих бізнесів втрачають ключові технології, коли йде досвідчений майстер. Документація — це страховка вашого бізнесу.

## Покрокова інструкція

### Крок 1: Картування знань (Knowledge Mapping)

```
Допоможи мені структурувати мій професійний досвід для навчання:

МОЯ СПЕЦІАЛЬНІСТЬ: [Напр: Електрик, Пекар, Плиточник]
ТЕМА ДЛЯ ПЕРЕДАЧІ: [Напр: Складання трифазних щитів]
РІВЕНЬ УЧНЯ: [Початківець / Середній / Профі]

ЗАПИТ:
1. ТЕОРІЯ: Які базові принципи треба знати? Які ДСТУ/ДБН регулюють цю роботу?
2. ПРАКТИКА: Покрокова послідовність дій (технологічна карта).
3. СЕКРЕТИ (Pro Tips): Те, чого не пишуть у підручниках (як зрозуміти, що суміш готова, як уникнути тріщин тощо).
4. ТИПОВІ ПОМИЛКИ: Що новачки роблять не так і як це виправити.

Збережи структуру у: ~/Cowork-Workspace/transfer/[тема]/mapping.txt
```

### Крок 2: Створення детальної інструкції (Procedure Sheet)

Для кожного критичного етапу створіть "Лист процедури":

```
Створи детальну інструкцію для: [Назва операції]

СТРУКТУРА ЛИСТА:
- МЕТА: Який результат ми хочемо отримати?
- БЕЗПЕКА: Які ЗІЗ (засоби захисту) потрібні? Які ризики?
- ОБЛАДНАННЯ: Список інструментів та матеріалів.
- КРОК ЗА КРОКОМ: 
  - Крок 1: Дія + Час + Як перевірити якість.
  - Крок 2: ...
- ЧЕК-ЛИСТ ЯКОСТІ: 5 пунктів для фінальної перевірки.

Формат: PDF для друку.
Збережи у: ~/Cowork-Workspace/transfer/procedures/[назва].pdf
```

### Крок 3: План навчання на 6 місяців

```
Розроби графік підготовки учня-плиточника на півроку:
- Місяць 1: Підготовка поверхонь, замішування розчинів, обрізка.
- Місяць 2: Укладання на стіни (прості схеми).
- ...
- Місяць 6: Складні вузли, кути 45 градусів, великий формат.

Додай критерії оцінки для кожного місяця.
```

### Крок 4: Створення візуальних підказок

ШІ може допомогти описати, як мають виглядати фото або схеми:

```
Опиши, які 5 ключових фото мені треба зробити на об'єкті, щоб створити візуальний гід для [Операція].
На що саме звернути увагу (ракурс, світло)?
```

### Крок 5: База знань бізнесу

```
Склади зміст "Книги Майстра" для моєї компанії. 
Включи туди: регламенти спілкування з клієнтом, правила чистоти на об'єкті, контакти перевірених постачальників, методику розрахунку матеріалів.
```

## Найкращі практики

1. **Записуйте голос** — Якщо вам важко писати, запишіть голосове повідомлення, описуючи свої дії під час роботи. Claude перетворить цей запис на професійну інструкцію.
2. **Фотографуйте помилки** — Краще один раз показати "як не треба", ніж довго пояснювати на словах.
3. **Метод "Тінь"** — Спочатку учень дивиться, потім ви робите разом, потім він робить під наглядом, і лише в кінці — самостійно.
4. **Оновлюйте дані** — Технології змінюються. Переглядайте ваші інструкції раз на рік (наприклад, у зимовий період, коли менше замовлень).
5. **Заохочуйте питання** — Питання новачків підсвічують дірки у ваших інструкціях. Додавайте відповіді на них у розділ FAQ.
