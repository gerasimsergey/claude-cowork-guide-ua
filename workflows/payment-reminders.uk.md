# Воркфлоу: Нагадування про оплату

> **Орієнтовний час**: 20 хвилин
> **Складність**: Середня
> **Категорія**: 📋 Адміністративні
> **Професії**: Підрядники, Торговці, Фрілансери, Малий бізнес

🌐 **Мови**: [English](payment-reminders.en.md) | [Français](payment-reminders.md) | [Українська 🇺🇦](payment-reminders.uk.md)

---

## Кейс використання

Клієнт не оплатив інвойс до встановленого терміну. Цей воркфлоу допоможе вам:

- Генерувати поетапні нагадування про оплату (ввічливе → наполегливе → офіційне)
- Законно розраховувати пеню за несвоєчасну оплату
- Готувати офіційне повідомлення (претензію) перед початком судових дій
- Відстежувати історію нагадувань про оплату

> ⚠️ **Важливо**: Завжди дотримуйтесь чинного комерційного законодавства та правил здійснення платежів. Пеня за несвоєчасну оплату повинна відповідати встановленим законом або договором ставкам.

### Крок 1: Ідентифікація простроченого інвойсу


### Крок 2: Генерація першого нагадування (R1 — Ввічливе)

```

- Клієнт: ABC Contracting Inc
- Сума: $1,500.00
- Термін оплати: 20 січня 2026 року
- Сьогодні: 5 лютого 2026 року (15 днів прострочення)

Згенеруй ВВІЧЛИВЕ нагадування про оплату (R1):

ТОН: Професійний, з розумінням, неконфліктний
ПРИПУЩЕННЯ: Випадкова забудькуватість або адміністративна затримка

ЗМІСТ:
- Посилання на номер та дату інвойсу
- Нагадування суми та терміну оплати
- Зазначення затримки (15 днів)
- Запит на підтвердження оплати
- Пропозиція допомоги в оплаті, якщо потрібно
- Надання реквізитів для оплати (банк, адреса для чеків)
- На цьому етапі без згадки про пеню

ФОРМАТ: Професійний діловий лист (PDF)
ТЕМА: "Нагадування про оплату — Інвойс INV-2026-042"

```

**Якщо відповіді немає через 7-10 днів** → Переходьте до Кроку 4

### Крок 4: Генерація другого нагадування (R2 — Наполегливе)

```

Сьогодні: 15 лютого 2026 року (25 днів прострочення)

Згенеруй НАПОЛЕГЛИВЕ нагадування про оплату (R2):

ТОН: Професійний, але більш суворий, нагадування про договірні зобов'язання
ПРИПУЩЕННЯ: Клієнт уникає оплати

ЗМІСТ:
- "Попри наше нагадування від 05.02.2026..."
- Нагадування деталей інвойсу (номер, дата, сума)
- Зазначення значного прострочення (25 днів)
- Згадка про договірні умови оплати
- Нарахування пені згідно з умовами договору:
  * Законна ставка: [вказати ставку, напр. облікова ставка + 10%]
  * Пеня: $1,500.00 × [ставка]% × 25 днів / 365 = $XX.XX
- Встановлення нового терміну оплати: 7 днів (22.02.2026)
- Попередження: "Невиконання зобов'язань призведе до надсилання офіційної претензії"
- Запит на письмове підтвердження

ФОРМАТ: Рекомендований лист (+ копія на email)

Зберегти до: ~/Cowork-Workspace/output/reminders/INV-2026-042-R2.pdf
```

```
Згенеруй ОФІЦІЙНУ ПРЕТЕНЗІЮ (R3):

ПРИПУЩЕННЯ: Ймовірна недобросовісність

ЗМІСТ:
ЗАГОЛОВОК: "ОФІЦІЙНА ПРЕТЕНЗІЯ — ФІНАЛЬНЕ ПОПЕРЕДЖЕННЯ ПЕРЕД СУДОВИМ ПОЗОВОМ"

- Офіційна вимога:
  * "Цим офіційно вимагаємо сплатити борг у сумі..."

- Згадка про:
  * Право на примусове стягнення через суд
```

## Усунення несправностей

### Клієнт оскаржує інвойс

**Причина**: Обґрунтована незгода з послугою або сумою
**Рішення**: Призупиніть нагадування, вирішіть суперечку:

```
Отримано заперечення щодо INV-2026-042:
Клієнт стверджує: "Послуга не виконана в повному обсязі"

НЕГАЙНА ДІЯ:
1. ЗУПИНИТИ всі нагадування
2. Створити досьє суперечки:
   - Оригінальний інвойс + кошторис
   - Листування з клієнтом
   - Докази надання послуги (фото, підписи, email)

ВИРІШЕННЯ:
Варіант А: Клієнт правий → Виписати кредит-ноту, оновлений інвойс
Варіант Б: Клієнт помиляється → Надати докази, провести переговори
Варіант В: Часткове заперечення → Коригувати інвойс, скласти план оплати

НЕ надсилайте офіційну претензію під час активної суперечки (ризик звинувачення в недобросовісності)
```

### Невпевненість щодо законних ставок пені

**Причина**: Ставка пені не вказана в договорі
**Рішення**: Застосуйте стандартну законну ставку:

```
Інвойс INV-2026-042 — У договорі не вказана ставка пені

ДОСЛІДЖУЙ застосовну ставку:
- Юрисдикція: [Штат/Країна]
- Стандартна ставка пені за комерційними платежами: [Пошук у базах даних]
- Приклад (США): Часто прайм-рейт + 1-3% або встановлена законом ставка
- Приклад (ЄС): Ставка ЄЦБ + 8 відсоткових пунктів

Розрахуй пеню за КОНСЕРВАТИВНОЮ ставкою, щоб уникнути суперечок:
"Пеня розрахована згідно з чинним законодавством: [Ставка]%"

Рекомендація: Додайте пункт про пеню в майбутні договори
```

### Клієнт ігнорує офіційну претензію

**Причина**: Недобросовісність або неплатоспроможність
**Рішення**: Оцініть доцільність судових дій:

```
Претензію надіслано 25.02.2026 — Термін 05.03.2026 минув
Ні відповіді, ні оплати

ДЕРЕВО РІШЕНЬ:

Сума > $2,000 ТА клієнт платоспроможний?
→ ТАК → Зверніться до адвоката для початку судового процесу
→ НІ → Оцініть співвідношення витрат та вигоди

Клієнт неплатоспроможний (банкрутство/закриття)?
→ Подайте заяву про визнання грошових вимог у процедурі банкрутства
→ Або спишіть як безнадійний борг (податкове вирахування)

Клієнт доступний, але відмовляється платити?
→ Медіація/арбітраж (якщо є відповідний пункт)
→ Суд малих позовів (якщо сума нижче порогу)
→ Колекторські послуги через адвоката

ВАЖЛИВО: Документуйте ВСЕ (усе листування, підтвердження доставок)
```

## Варіації

### Пропозиція плану оплати

```
Клієнт відповідає на R2: "Фінансові труднощі, не можу сплатити всю суму зараз"

Згенеруй пропозицію ПЛАНУ ОПЛАТИ:
- Основний борг: $1,500.00
- Пеня скасовується, ЯКЩО план буде дотримано
- План: 3 щомісячні платежі по $500 (15.03, 15.04, 15.05)
- Умови:
  * Перший платіж = підтвердження намірів (обов'язково)
  * Пропуск платежу = негайна виплата всієї суми + пені
  * Потрібна письмова угода (обидва підписи)

Юридичне застереження: "Цей план не є прощенням боргу..."

PDF: ~/Cowork-Workspace/output/agreements/INV-2026-042-payment-plan.pdf
```

### Експрес-претензія

```
Велика сума ($10,000+) або термінова ситуація

Згенеруй ЕКСПРЕС-претензію (термін 5 днів):
- Прискорені терміни через значну суму
- Згадка про ризик значних додаткових витрат
- Фірмовий бланк адвоката (якщо є)
- Заява про намір накласти арешт на майно/активи, якщо актуально
- Запит на негайний контакт для вирішення питання

Рекомендований лист + email + кур'єрська доставка для простежуваності
```

### Нагадування з пропозицією врегулювання (Settlement)

```
Старий борг (6+ місяців), бажання швидко закрити справу

Згенеруй R2 з пропозицією врегулювання:
- Оригінальна сума: $1,500.00
- Нарахована пеня: $150.00
- ПРОПОЗИЦІЯ: "Сплатіть $1,400 протягом 10 днів = повне виконання зобов'язань"
- Дисконт: $250 (скасування пені + $100 знижка доброї волі)
- Умови: Тільки повна оплата (без розстрочки), тільки банківський переказ

Вигода: Швидке отримання коштів замість тривалої суперечки
Ризик: Створює прецедент для інших клієнтів

Вкажіть: "Разова виняткова пропозиція, що не підлягає повторенню"
```

## Найкращі практики

1. **Послідовність ескалації** — Дотримуйтесь інтервалів між рівнями (R1 +10 днів → R2 +7 днів → R3)
2. **Завжди ввічливо** — Навіть R3 залишається професійним, уникайте образ чи погроз
3. **Юридична відповідність** — Перевіряйте застосовні ставки та закони про стягнення боргів у вашій юрисдикції
4. **Документуйте все** — Зберігайте копії всіх нагадувань та підтверджень вручення
5. **Обирайте пріоритети** — Судові дії коштують грошей; оцініть, чи варта справа витрат
6. **Сприяння оплаті** — Пропонуйте плани оплати перед офіційною претензією (це підвищує шанси на повернення)
7. **Рекомендована пошта** — R2 і особливо R3 слід надсилати рекомендованим листом
8. **Систематичне відстеження** — Використовуйте таблицю для моніторингу всіх кейсів (щоб нікого не забути)
9. **Консультація адвоката** — Для сум понад $5,000 або складних ситуацій зверніться до адвоката
10. **Профілактика** — Додавайте чіткі умови оплати та пункти про пеню у ваші договори
