# Шаблони Email (Email Templates)

🌐 **Мови**: [English](email-templates.en.md) | [Français](email-templates.md) | [Українська 🇺🇦](email-templates.uk.md)

Цей воркфлоу дозволяє Claude Cowork виступати в ролі вашого копірайтера, створюючи набір готових до використання шаблонів листів для різних бізнес-ситуацій.

---

## 📋 Контекст

Ви витрачаєте забагато часу на написання типових листів (вітання нових клієнтів, відповіді на запити ціни, пояснення умов роботи). Вам потрібна база якісних шаблонів, які легко адаптувати.

## 🛠️ Підготовка

1. Підготуйте список ситуацій, для яких потрібні шаблони.
2. (Опціонально) Надайте приклади ваших найкращих минулих листів, щоб Claude вивчив ваш стиль.

---

## 📝 Промпт (Запит)

Скопіюйте цей запит у Claude:

```text
CONTEXT: I need to standardize my client communications for MacPlus Software.
TASK: Based on my business description in 'about_us.md', create 5 professional email templates:
1. Welcome email for new clients.
2. Follow-up after a first meeting.
3. Polite request for missing information.
4. Explaining a delay in project delivery.
5. Asking for a testimonial/review after project completion.
OUTPUT: Save these templates in a Markdown file 'Email_Templates_Library.md'.
CONSTRAINTS: Use a 'Friendly yet Professional' tone. Use placeholders like [Client Name] and [Project Name].
```

---

## ⚙️ План виконання

1. **Аналіз стилю**: Claude вивчить ваш "голос бренду" та специфіку послуг.
2. **Генерація**: Система створить тексти, які звучать природно, а не як автоматичні розсилки.
3. **Структурування**: Ви отримаєте організовану бібліотеку, звідки зможете копіювати тексти у ваш поштовий клієнт.

---

## ✅ Перевірка результату

- Перевірте, чи не звучать шаблони занадто офіційно або навпаки — занадто фамільярно.
- Переконайтеся, що всі плейсхолдери (місця для заповнення) чітко виділені.
- Спробуйте адаптувати один шаблон під реального клієнта — чи легко це зробити?

---

## 💡 Поради

- **Мультиязичність**: Ви можете попросити Claude створити ці ж шаблони англійською мовою для роботи з міжнародними клієнтами.
- **Відповіді на заперечення**: Створіть окрему групу шаблонів для ввічливої відмови або пояснення, чому ваші послуги коштують дорожче, ніж у конкурентів.
